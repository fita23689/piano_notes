## 📖 笔记特点

每节笔记约 18 KB / 400+ 行，采用统一结构：

| 模块 | 说明 |
|---|---|
| **本课概览表** | 5 秒掌握本节学了什么 |
| **核心知识章节** | 概念 → 视觉化 → 比喻 → 实战 |
| **键盘 / 谱面图示** | 用 ASCII 字符还原视频中的关键画面 |
| **Andrew 金句** | 抽出原视频里令人印象深刻的英文句子 + 中文翻译 + 时间戳 |
| **练习清单** | 必做 / 进阶 / 听力训练分级 |
| **🔬 双校准** | 与通用乐理 / 钢琴知识对照，纠正视频中的口误 + 补充遗漏 |
| **总评打分** | 知识准确度 / 演示清晰度 / 信息密度 / 适合初学者 |

## 🎯 双校准：本笔记的灵魂

不同于"翻译型笔记"，每节都做了**与通用钢琴/乐理知识的对照**，主动标注：

- ✅ **完全正确**的内容
- ⚠️ **表达瑕疵 / 易混淆点**
- ❌ **视频里的事实错误**（已逐条纠正）
- 💡 **视频遗漏的重要内容**（已补充）

### 累计纠正示例（10+ 处）

| 课 | 视频说法 | 纠正 |
|---|---|---|
| ep04 | 3/4 是 compound time | 实为 simple triple |
| ep05 | "Green Boy" | 标准是 "Good Boy" |
| ep07 | 音板用 pine（松木） | 实为 spruce（云杉） |
| ep09 | "inharmonic keys" | 应为 enharmonic |
| ep09 | G♭ 大 ≡ F♯ 小 | 应为 G♭ 大 ≡ F♯ 大 |
| ep14 | 3/4 又被叫为 compound | 实为 simple |
| ep24 | 节拍器发明者 | 修正历史信息 |
| ep27 | Imperfect Cadence | 应为 Half Cadence |
| ep28 | Tonalization | 应为 Tonicization |
| ep29 | Für Elise 的 Op.编号 | 实为 WoO 59 |

## 🚀 推荐学习路径

### 全新初学者（按顺序学）

```
第 1-3 课：键盘 / C 大调 / 三和弦                  ← 基础
第 4-5 课：节奏 / 五线谱                          ← 识谱
第 6 课  ：音程 + 力度 + 临时记号                 ← 杂项
第 7 课  ：钢琴构造（选学）                       ← 知识扩展
第 8-10 课：小调 / 五度循环 / 半音阶              ← 调性体系
第 11+ 课：进入实战 + 真曲                        ← 深化
```

### 已有基础的人（挑感兴趣的看）

- **想攻克识谱**：第 5 + 第 16 + 第 19 课
- **想理解和声**：第 6 + 第 15 + 第 27 + 第 30 课
- **想练技术**：第 10 + 第 21 + 第 32 + 第 34 + 第 35 课
- **想上台**：第 23 + 第 36 课
