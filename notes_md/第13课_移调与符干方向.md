## 2. 移调（Transposition）：核心概念

[00:11] *"Transposition is basically taking something, moving it up or just moving it down."*

### 定义

> 🎯 **移调 = 把整段乐曲从一个调"搬"到另一个调，保持旋律的相对走向不变**

### 直观感受

```
G 大调原曲：     C 大调移调：    F 大调移调：
[音1] [音2] ...  [音1↑] [音2↑]   [音1↓] [音2↓]
        ↓
保持"上 3 下 2 上 4"等相对距离不变
   ↓
听起来还是同一首歌，只是"高了"或"低了"
```

### 为什么要移调？

| 场景 | 原因 |
|---|---|
| **歌手嗓音不合适** | "原曲太高了，我唱不上去" → 整体下移 |
| **乐器音域限制** | 长笛弹不到 G2 → 整曲上移 |
| **简化指法** | F♯ 大调（6 升号）太复杂 → 移到 G 大调（1 升号） |
| **配合主题转换** | 流行歌副歌时常**整体上移半音 / 全音**（常见在最后副歌）|

## 3. 移调的两种方法对比

### 方法 A：逐音换名（最笨）

```
G 大调原音：G  A  B  G  E  D
            ↓ 一个一个查"移到 C 大调对应什么音"
C 大调结果：C  D  E  C  A  G
```

→ **缺点**：每次都要重新查表，慢且容易出错。

### 方法 B：用音程换算（推荐！）

[03:34] *"This is where intervals help."*

```
关键洞察：旋律的"音程关系"在每个调里都不变！
                ↓
1. 找出原曲每两个相邻音的"音程"
2. 在新调里，按同样的音程关系往上/下走
3. 加上新调的调号（升降号自动到位）
```

#### 实战例子

```
G 大调原音：G  A  B  G    
音程关系：     +2  +3  -3   （+ 是上行，- 是下行；数字是度数）

C 大调移调：从 C 起步
             ↓ +2 度（上 2 度）
             D
             ↓ +3 度（上 3 度）
             F  ← 注意是 F，不是 E！（C 起 +3 度是 E，但 D 起 +3 度是 F）
             ↓ -3 度（下 3 度）
             D
             
等等！这里要小心，应是从 G 算起的相对音程
```

> ⚠️ 上面只是说明思路。**实际推荐做法**：用 Andrew 的"主音对齐法"。

### 方法 C：主音对齐法（最简单！）

[02:55] *"G is the tonic note in G major. C is the first note in C major. So I just move it up."*

#### 思路

```
原调主音 → 新调主音 = 整体平移距离
                ↓
         所有音都按这个距离平移
```

#### 实战

```
G 大调主音 = G
C 大调主音 = C
G → C = 上行 4 度

所以原曲所有音都"上行 4 度"
   ↓
G 大调原音：G  A  B  G  E  D
平移 +4 度：C  D  E  C  A  G  ← 这就是 C 大调版本！
```

#### 为什么这么简单？

因为**G 大调 → C 大调**是"主音对齐"，只要找到原调主音和新调主音的距离，整段平移就 OK。

## 4. 实战：从 G 大调移到 C 大调

[03:34] Andrew 的视频示范。

### 原曲（G 大调，4/4 拍）

```
G 大调原谱（节奏忽略）：
G  B  A  G   ← 上 3 下 2 下 2
F♯ G  A  B  G  ← 各种短音程
```

### 第 1 步：定主音对齐距离

```
G → C = 上行 4 度
（即：每个音都向上数 4 个音名）
```

### 第 2 步：逐音平移

```
G 大调  →  +4 度  →  C 大调
G            →    C
B            →    E（B 上 4 度 = E）
A            →    D
G            →    C
F♯           →    B（F♯ 上 4 度 = B，但 F♯ 上 4 度严格是 B，无升号）
G            →    C
A            →    D
B            →    E
G            →    C
```

### 第 3 步：套用新调号

```
C 大调没有升降号
   ↓
所有 F 都是 F（不需要 F♯）
所有其他音也保持自然
```

### 第 4 步：验证

```
C  E  D  C  B  C  D  E  C
   ↓ 听起来和原曲"形状一样"，只是高了 4 度
   ↓
✅ 移调成功
```

### Andrew 的另一个示范：移到 F 大调

[03:48] *"Now I'm going to play F major."*

```
G → F = 下行 2 度（往下数 2 个音名）

G 大调原音： G  B  A  G  F♯ G  A  B  G
平移 -2 度： F  A  G  F  E  F  G  A  F

但 F 大调有 1 个降号 (B♭)，所以要用 B♭ 替换 B
   ↓
最终： F  A  G  F  E  F  G  A  F
（无 B 出现，所以 B♭ 调号没用上）
```

## 5. 移调流程总结：4 步法

> 🎯 **必背流程**！

### 4 步法

```
1. 算距离  ：原调主音 → 新调主音的音程（上行/下行 几度）
2. 整体平移：每个音都按这个距离移动
3. 套调号  ：根据新调的调号自动加升降号
4. 检查听感：弹一弹确认"形状一样"
```

### Andrew 的核心洞察

[04:54] *"Find the distance intervalically and move it up or down to match that."*

→ **用音程思维做移调**，比逐音查找快 10 倍。

### 难度梯度

| 移调任务 | 难度 |
|---|---|
| C 大 → G 大（移 +5 度） | ⭐ 简单 |
| C 大 → F 大（移 -5 度 = +4 度） | ⭐ 简单 |
| C 大 → D 大（移 +2 度） | ⭐⭐ 中 |
| C 大 → F♯ 大（移 +6 度） | ⭐⭐⭐⭐ 难（6 个升号） |
| C 大 → c 小（同主音平行调） | ⭐⭐⭐ 中（要换调式） |

### 进阶：调式移调（同主音平行）

```
C 大 → c 小：主音不变，但调式从大调变小调
            ↓
         不是真正的"移调"，而是"换调式"
         所有 E 变 E♭，A 变 A♭，B 变 B♭
```

→ 这种"明转暗"的处理在浪漫派音乐中超常见（如肖邦的"明 → 暗"对比）。

## 6. 符干方向（Stem Direction）

[06:00] **本课第 2 大主题**：为什么乐谱上有的符干朝上，有的朝下？

### 符干（Stem）是什么？

```
⚪+丨   ⚫+丨   ♪
↑     ↑     ↑
都有"丨"，这就是符干
```

> 💡 复习：第 4 课讲过音符的形状。符干 = 音符头上的那根直棍。

## 7. "指南针法则"：朝中线指

[07:21] *"Think of this note as a compass."*

### 核心规则

> 🎯 **符干永远指向五线谱的中线（第 3 线）**

### 视觉化

```
═══════════ 第 5 线
                       ⚫(高音)
═══════════ 第 4 线   丨   ← 符干朝下，指向中线
                       
═══════════ 第 3 线 ← 中线（"磁极"）
                       
                       丨   ← 符干朝上，指向中线
═══════════ 第 2 线   ⚫(低音)

═══════════ 第 1 线
```

### 规律

| 音的位置 | 符干方向 |
|---|---|
| 中线**以上** | 符干**朝下**（向中线"伸") |
| 中线**以下** | 符干**朝上**（向中线"伸"） |
| 正好在中线（B 在高音谱号 / D 在低音谱号） | **任意方向**（看上下文） |

### 适用范围

[07:43] *"It doesn't matter if it's the bass clef or the treble clef."*

→ **高音谱号、低音谱号都用同一规则**！只是"中线"在两个谱号上对应**不同的音名**：
- 高音谱号中线 = **B**
- 低音谱号中线 = **D**

## 8. "拔河法则"：多音组合时谁赢

[08:43] **当多个音用同一个符干（如八分音符 / 十六分音符）连起来时**，符干方向怎么定？

### Andrew 的"拔河"比喻

[09:01] *"It has to have a battle. Like this group has to come to a decision."*

### 规则 1：看哪个音"离中线最远"

```
两个音连一起：A（间 5）+ E（线 1）

A 离中线（第 3 线）距离 = 1 个间 = "1 单位"
E 离中线（第 3 线）距离 = 2 条线 = "2 单位"
                                   ↑
                            E 离中线更远 → E 决定方向
                            E 在中线下 → 符干朝上
```

### 规则 2：看"多数"

```
四个音组：C, A, A, C   （C 在第 3 间附近，A 在第 2 间）
                      ↓
             两个 C ≈ 中性（在中线附近）
             两个 A ≈ 中线下
                      ↓
             "多数"在中线下 → 符干朝上
```

### 规则 3：完全平局时随意

```
一组：A（中线上 1 单位）+ A（中线下 1 单位）= 完全对称
                                          ↓
                              符干上下都对，按上下文好看选
```

### Andrew 的"婚姻"比喻

[09:11] *"It's kind of like a marriage where the stronger person wins the argument."*

### 实战识别流程

```
1. 找出组内所有音
2. 分别量"距中线"的距离
3. 离中线最远的那个 → 决定方向（或按"多数"原则）
4. 平局 → 按上下文好看
```

## 9. 多声部记谱（Choral Style Writing）

[13:13] **打破常规符干规则的特殊情况**：当一个谱表里要写**两个独立声部**时。

### 什么是多声部？

```
一个谱表里同时写着两条独立旋律线，分别有自己的节奏和符干方向
                ↓
           听起来像两个乐器在合奏
           （但实际只用一架钢琴弹）
```

### 例子：四部和声（SATB）

```
经典合唱写法：
- 高音谱表 → 写 Soprano（女高）+ Alto（女中）
- 低音谱表 → 写 Tenor（男高）+ Bass（男低）
                   ↓
           4 个独立声部，互相协调
```

### 钢琴上的多声部

```
钢琴谱里两个声部常见于：
- Bach 复调作品（赋格）
- 古典奏鸣曲的中间声部
- 浪漫派的"主旋律 + 内声部"
```

## 10. 多声部的"上下双向符干"

### 写法规则

[13:46] *"Think of this as completely independent of this."*

```
高音谱表里：
              ⚫
   ⌐丨        ┘  ← Soprano 声部，符干朝上（无论高低）
═══════════
   丨⌐  ⚪    
              丨⌐  ← Alto 声部，符干朝下（无论高低）
═══════════
```

### 核心规则

> 🎯 **每个声部的所有音，符干方向一致**！
> - 上声部：永远朝上
> - 下声部：永远朝下

→ 这**违反**了第 7 节的"指南针法则"，但**符合多声部的可读性**。

### Andrew 的演示

[14:42] 他在视频里画了一个例子：上声部是连续的二分音符（朝上），下声部是各种节奏（朝下）。两个声部**节奏完全独立**：
```
上声部（朝上）：⚪__________________⚪__________________
下声部（朝下）：⚫⚫_⚫⚫⚫⚫_⚫⚫⚫⚫⚫⚫_
```

→ 即使一个声部"长音"，另一个声部也能"独立运动"。

### 共用同一个音符的特殊情况

[15:51] *"If you had two violins like violin 1 and violin 2 playing the same note, you'd have one note with two stems."*

```
两个声部偶尔会汇合在同一音上：
           ⌐丨↑      ← 上声部符干朝上
       ⚫
           ⌐丨↓      ← 下声部符干朝下
                ↓
   一个音符头 + 两个上下双向符干
   
   表示：两个声部"恰好同时"弹这个音
```

→ 演奏时**只弹一次**这个音，但**记谱上要标双符干**以表明两个声部都在这里"汇合"。

### 何时会用到？

| 风格 | 多声部记谱例子 |
|---|---|
| **巴洛克赋格** | 巴赫《十二平均律》，每个谱表 2-3 个声部 |
| **古典奏鸣曲** | 莫扎特、贝多芬中速段，常有"主旋律 + 内声部" |
| **浪漫钢琴曲** | 肖邦夜曲、舒曼《童年情景》|
| **流行编曲** | 钢琴左手的"和弦 + 低音线"分两声部 |

## 11. 练习清单

### 必做（每周 2-3 次）

- [ ] **C 大调一段简单旋律**（如《小星星》前 4 小节），分别移到 G 大、F 大、D 大。
- [ ] **音程检查**：每个移调结果，验证音程关系是否保持。
- [ ] **看谱认符干**：拿一首巴赫小曲，逐音判断"为什么这个符干朝上 / 下"。

### 进阶

- [ ] **同主音平行调**：把 C 大调《小星星》改写成 C 小调（换调式）。
- [ ] **复杂调移调**：尝试把曲子从 D 大调（2 升）移到 A♭ 大调（4 降）。
- [ ] **多声部识别**：找一首巴赫两声部小步舞曲，圈出每个声部。

### 听觉训练（选做）

- [ ] **听辨移调**：让朋友把同一段旋律弹两遍（不同调），听出移调距离。
- [ ] **多声部听辨**：听巴赫赋格，努力分辨**同时听到的 2-3 条独立旋律**。

## 13. 双校准：与通用乐理对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **移调定义** —— 标准。
2. **用音程做移调** —— 正确，是专业演奏者的标准做法。
3. **主音对齐 = 整体平移** —— 正确（这是"平行移调"的基本原理）。
4. **新调要套新调号** —— 正确。
5. **符干指向中线（第 3 线）** —— 标准记谱规则。
6. **多个音连起来时按"远离中线"原则定方向** —— 正确。
7. **多声部记谱（上声部朝上，下声部朝下）** —— 标准 SATB 写法。
8. **同一音符的双符干表示两个声部汇合** —— 正确。
9. **巴洛克 / 浪漫派的多声部使用** —— 历史属实。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[03:18] "C is the first note in C major"** —— Andrew 想说"主音"，但口误说成"first note"。  
   → **澄清**：**主音 (tonic) = 调的第一个音，也是调名命名的音**。两者一回事。

2. **[03:50] "Going from C down to F, which is down a fifth"** —— ⚠️ 严格来说从 C 到 F 是**下行 5 度** 或 **上行 4 度**，两个说法都对（互补音程）。Andrew 没说错，但视频里的措辞略乱。

3. **[06:30] "Compass" 比喻 + Andrew 跑题讲指南针不灵** —— 与教学无关，本笔记跳过。

4. **[10:15] "Two down here and one up there"** —— Andrew 在数音的"远近"时用了"个数"，但实际**距离 + 个数**都重要：
   → **完整规则**：先看"离中线最远的音"决定方向；如果距离相同，再看"多数"。

5. **[15:51] "Both voices are playing that note... still play it the same"** —— Andrew 解释清楚了：**听感**上一次，**记谱**上两个符干。

### ❌ 视频中的小错误

1. **[00:53] Andrew 忘记写拍号** —— 自嘲式失误，无关教学。

2. **关于符干方向的"完整规则"**：Andrew 简化成了"看离中线远 + 看多数"，但更专业的规则还包括：
   - **符干长度**：标准是**一个八度**（不是任意长）
   - **符尾连接**（beam）方向：与符干一致
   - 这些细节视频没讲，但对**写谱**很重要。

### 💡 视频遗漏的重要内容

1. **移调的"移调乐器"概念**：B♭ 单簧管、F 圆号等都是**移调乐器**，它们看的"C"实际响"B♭"或"F"。这种"乐器内置移调"在管弦乐中非常重要，视频没提（钢琴课不必讲，但值得了解）。

2. **移调的常见目的细化**：
   - **歌手适配**：流行歌手最常用，每个人音域不同。
   - **吉他变调夹（Capo）**：吉他手"假装移调"用具，原理类似。
   - **自动伴奏机**：电子琴上有"transpose"按钮一键移调。

3. **难调的简化技巧**：F♯ 大调（6 升号）和 G♭ 大调（6 降号）听起来一样，**记谱却不同**。专业演奏家根据"周围调"决定用哪种写法。

4. **符干与"乐谱印刷规范"**：现代乐谱印刷有更细的规定（如符干粗细、符尾间距），但视频简化讲了核心。

5. **多声部的"独立性"原则**：
   - 每个声部的**节奏**可以完全独立
   - 每个声部的**力度记号**可以独立
   - 每个声部的**连音线、断奏**也独立
   → 视频简单提了节奏独立，没说清楚每个声部都是"独立单位"。

6. **数字移调**：现代电子琴有"transpose -1, +1, +2..."等简单数字调档。**功能等同于音程移调**，只是更直观。
