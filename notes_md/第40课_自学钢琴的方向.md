## 3. 第三阶段：找老师 + 投资

[03:45] **Andrew 的强烈建议**！

### 老师的"不可替代性"

[03:48] *"You can't really go that much further without a teacher."*

```
为什么必须找老师？
1. 个人**反馈**（视频不能）
2. **纠正**坏习惯（视频不能精确）
3. **激励**督促（视频不能）
4. **个性化**进度（视频是通用）
5. **真人**演示（视频角度有限）
        ↓
    没老师 = 90% 的人**会停留**
            ↓
    经验数据
```

### 找老师的"3 大类型"

```
类型 1：业余老师
- 价格：$30-60/小时
- 适合：业余兴趣
- 教法：基础

类型 2：专业老师
- 价格：$60-150/小时
- 适合：考级 / 比赛
- 教法：严格 + 系统

类型 3：大师级
- 价格：$200-500/小时
- 适合：职业准备
- 教法：极严 + 个性
        ↓
    根据**目标**选老师
```

### Andrew 的"性格匹配"建议

[04:21] **重申第 35 课的话**：

```
找老师 = 找"音乐伴侣"：
- 性格匹配
- 价值观一致
- 教学方法适合你
- 互相**化学反应**好
            ↓
    一年面试 5-6 个
            ↓
    不合适立即换
```

### 价格 vs 价值

[04:50] *"There's no real way to get to professional level without spending money."*

```
事实：
- 业余 = 免费 / 便宜
- 中级 = 中等投资
- 专业 = **巨大**投资
        ↓
    估算：
    - 10 年专业训练：$30,000-100,000
    - 包括：老师 + 钢琴 + 比赛 + 大师课
            ↓
    "天才钢琴家"几乎都有**家庭资助**
```

## 6. 互联网学习的优势与局限

[14:30] **整集第二大主题**！

### Andrew 的"互联网情结"

[15:02] *"I'm really trying to do a different way of teaching by using the Internet."*

```
Andrew 的初心（2008 年）：
- 让**所有人**学得起钢琴
- 用 YouTube 革命**钢琴教学**
- 免费 + 全球
            ↓
    14 年后（2022）：
    频道**500 万**订阅
    全球数百万学生**自学**
            ↓
    Andrew 的梦想**实现**了
```

### 互联网的"4 大优势"

```
优势 1：免费 / 极低价
- 老师 1 小时 $50
- YouTube 1 小时 $0
        ↓
优势 2：可随时学
- 不用预约
- 自己安排
        ↓
优势 3：可重看
- 老师讲一次
- YouTube 看 100 次
        ↓
优势 4：全球共享
- 印度小村庄孩子
- 也能学 Andrew 课
        ↓
    "音乐民主化"
```

### 互联网的"5 大局限"

[15:33] **诚实**：

```
局限 1：无个人反馈
- 视频不能听你弹
- 不能纠正你

局限 2：无适应性
- 视频是**通用**的
- 不能"个性化"

局限 3：质量参差
- 99% 视频**烂**
- 1% 才有用

局限 4：动力不足
- 没人督促
- 容易放弃

局限 5：技术细节缺失
- 复杂技术**必须**真人示范
- 视频角度限制
        ↓
    无法**完全**替代老师
```

### Andrew 的"互联网未来"

[15:25] **预测**：

```
"There's so many videos online for so many different things and they're really really good. They're just as good or even better than going to a teacher."
        ↓
    Andrew 预测（2008）：
    - 互联网将**超越**老师
    - 但需要**多年**发展
            ↓
    14 年后（2022）：
    部分实现，但**老师仍重要**
```

### 现代视角（2026）

```
2026 年的"AI + 互联网":
✅ ChatGPT 答钢琴问题
✅ AI 评价你的演奏
✅ 在线大师课（Zoom）
✅ MIDI 实时反馈
✅ 录像云端储存
✅ Discord 学习社群
        ↓
    比 2008 年**强 100 倍**
            ↓
    但**老师 + 现场**仍最有效
```

## 11. 毕业后练习清单

### 第一周：消化

- [ ] **重看** 5 节最难的（你自选）
- [ ] **完整弹** 1 首学过的曲子
- [ ] **录视频** 自我评估
- [ ] **写笔记**：哪里弱？

### 第一个月：决定方向

- [ ] **思考**：兴趣 / 进阶 / 专业？
- [ ] **找老师**（如选进阶 / 专业）
- [ ] **设定目标**：3 个月 / 6 个月
- [ ] **建立规律**：每天 X 分钟

### 前 3 个月：建立习惯

- [ ] **每天练习** 15-60 分钟
- [ ] **每周 1 首新曲**
- [ ] **每月录视频**
- [ ] **加入社群**（Reddit / Discord）

### 6 个月：进入"中级"

- [ ] **完成 5-10 首中等曲**
- [ ] **参加业余比赛 / 演出**
- [ ] **找老师**（如未找）
- [ ] **学新技术**（Andrew 没讲的）

### 12 个月：评估

- [ ] **录"对比视频"**：12 月前 vs 现在
- [ ] **决定**：继续 / 提升 / 放慢
- [ ] **长期计划**：2-3 年
