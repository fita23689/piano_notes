## 2. 大调 vs 小调：明亮 vs 忧郁

[01:23] *"The major is the happy sounding, and the minor is the sad sounding."*

### 听感对比

| 调式 | 听感 | 用途 |
|---|---|---|
| **大调 (Major)** | 明亮、欢快、积极 | 圣诞颂歌、流行欢快曲、婚礼进行曲 |
| **小调 (Minor)** | 阴郁、悲伤、神秘 | 哀歌、悬疑配乐、深沉抒情 |

### 经典例子

| 大调名曲 | 小调名曲 |
|---|---|
| 《Let It Be》（C 大调）| 《Adagio for Strings》（B♭ 小调）|
| 《欢乐颂》（D 大调）| 《月光奏鸣曲第一乐章》（C♯ 小调）|
| 《卡农》（D 大调）| 《Greensleeves》（A 小调）|

### 历史背景

[00:46] **教会调式（Church Modes）**：在大调/小调系统出现之前（中世纪），西方音乐用 7 种"教会调式"（Ionian、Dorian、Phrygian 等）。  
**等比律（Equal Temperament）**：现代钢琴的调律方式，让 12 个半音平均分配，每个调都能转。  
→ 大调/小调系统就是基于等比律发展出来的。

[00:57] *"They developed this whole new system... having a major scale and a minor scale for each key."*

## 3. 关系大小调（Relative Major/Minor）

[01:41] *"C major and C minor are not related at all. C major is actually related to A minor."*

### 关键概念：**共享同一个调号**

```
C 大调：没有升降号       │
                        ├→  调号相同 = 关系调
A 小调：没有升降号       │
```

### 寻找规则

> 🎯 **从大调找小调**：往**下**数 **3 个半音**。  
> 🎯 **从小调找大调**：往**上**数 **3 个半音**。

### 助记口诀

[14:03] *"If you're sad in a minor key, you have to go up to find the happy. If you're in a happy major, you have to go down to find the sad."*

> 😢 小调（悲伤） → 想要快乐 → **往上 3 半音** → 大调（快乐）  
> 😊 大调（快乐） → 想要悲伤 → **往下 3 半音** → 小调（悲伤）

### 实战推导

| 大调 | 往下数 3 半音 | 关系小调 |
|---|---|---|
| C 大调 | C → B → B♭ → A | **A 小调** |
| G 大调 | G → F♯ → F → E | **E 小调** |
| D 大调 | D → C♯ → C → B | **B 小调** |
| F 大调 | F → E → E♭ → D | **D 小调** |
| A♭ 大调 | A♭ → G → G♭ → F | **F 小调** |

### Andrew 的"小测验"

[14:50] **问题**："C 小调的关系大调是什么？"  
**答案**：C 小调 → 上 3 半音 → C → C♯ → D → **E♭** → **E♭ 大调**！

→ 验证：E♭ 大调有 3 个降号（B♭, E♭, A♭）；C 小调也有 3 个降号 ✓

## 4. 自然小调（Natural Minor）

[03:21] *"It's just the same as C major but three semitones lower. That's it."*

### 定义

**和关系大调用同一个调号，从小调主音开始，往上弹 8 个音**。

### A 小调（自然）= 全部白键

```
A   B   C   D   E   F   G   A
↑                           ↑
小调主音                  高 8 度回归
```

[04:38] *"There's no sharps or flats, all white."*

### 全音 / 半音公式

```
T - S - T - T - S - T - T
全 半 全 全 半 全 全
A→B B→C C→D D→E E→F F→G G→A
```

→ 与大调的 `T-T-S-T-T-T-S` **完全不同**！

### 自然小调的"问题"

[06:23] 自然小调的最后两个音 G→A 是"全音"，**听起来不够"想回家"**。  
→ 缺乏向主音的"牵引力" → 听起来悬而未决。

[05:00] *"Natural minor is very rarely used."*  
（自然小调实际演奏中**用得很少**！原因下一节解释。）

## 5. 导音（Leading Tone）

[05:14] *"There's something called a leading tone."*

### 什么是导音？

> 🎯 **导音 = 音阶的第 7 个音 = 距离主音半音的那个音**

### 为什么叫"导音"？

[05:25] *"It pulls you back to the tonic."*  
→ 它**牵引你回到主音**！像有磁力一样想"回家"。

### 验证：在 C 大调中

```
C  D  E  F  G  A  B  C
1  2  3  4  5  6  7  1
                  ↑
              导音 = B
              B → C 只有半音 → 强烈牵引感
```

试试弹：`C D E F G A B`，停在 B，**会非常想再弹一个 C**！

### 自然小调缺什么？

```
A  B  C  D  E  F  G  A
1  2  3  4  5  6  7  1
                  ↑
              第 7 音 = G
              G → A 是全音 → 牵引感弱！
```

→ 这就是为什么自然小调"不够过瘾"。

## 6. 和声小调（Harmonic Minor）

[04:57] *"Harmonic minor is probably the most commonly used."*

### 修改规则

> **把自然小调的第 7 音升高半音**，制造出"半音导音"。

### A 和声小调

```
A   B   C   D   E   F   G♯   A
                        ↑
                    升号 ← G 升到 G♯
```

### 全音 / 半音公式

```
T - S - T - T - S - 增二度 - S
全 半 全 全 半  +1.5 半 半
                    ↑
                这里是个增二度（augmented 2nd）= 1.5 全音
```

### 听感特点

[07:14] *"This sounds a little bit like Arabian music."*

→ 因为 F → G♯ 是 **增二度**（augmented second），听起来像中东 / 西班牙弗拉门戈 / 印度音乐风。

### 名字由来

[12:08] *"They call it harmonic minor because it was used in harmony for chords."*

→ 升高第 7 音是为了**和弦进行**：让属和弦（V）能解决到主和弦（i）。

### 经典进行示例

```
i (Am: A-C-E)  →  V (E大: E-G♯-B)  →  i (Am: A-C-E)
                       ↑
                  这里的 G♯ 来自和声小调
                  否则就是 v (Em: E-G-B) 没那么有牵引力
```

### Andrew 的搞笑比喻

[06:48] *"It's like they put the natural scale on steroids: 'I'm beefy and buffy, and I will push you to the tonic!'"*

→ 把自然小调"打了类固醇"，强化了向主音的牵引力。

## 7. 旋律小调（Melodic Minor）

[08:34] **最复杂的一种**，规则会因为"上行 / 下行"而变化。

### 背景：中世纪教会的禁忌

[08:43] *"In the Middle Ages, they outlawed the interval of an augmented second."*

→ 教会认为**增二度（augmented 2nd）听起来"邪恶"**，不许写！

### 但和声小调里就有增二度啊（F → G♯）！

[09:42] *"They had to write around it."*

→ 解决方案：**把 F 也升高半音**，变成 F♯，消除增二度。

### A 旋律小调（上行）

```
A   B   C   D   E   F♯   G♯   A
                    ↑    ↑
                  升 6   升 7
```

### A 旋律小调（下行）

[10:50] *"On the way down, you lower the notes."*

```
A   G   F   E   D   C   B   A
↑   ↑   ↑
还原（变回自然小调）
```

→ **下行时把第 6 和第 7 音都还原成自然小调的版本**！

### 完整的 A 旋律小调

```
上行：A   B   C   D   E   F♯   G♯   A
                          ↑    ↑
                        升号  升号

下行：A   G   F   E   D   C   B   A
      ↑   ↑   ↑
    （没升号，全部回到自然小调）
```

### 名字由来

[16:53] *"Melodic scales are usually used for melody."*

→ 旋律小调用于**旋律线条**（一串单音），所以下行时不需要导音的牵引力，可以用更"自然"的版本。

### Andrew 的总结口诀

[17:21] **"Six and seven raised when going up, both lowered when going down."**

> 🎯 **6 升 7 升上去，6 还原 7 还原下来**

## 8. 三种小调的对比总表

> 🎯 这是本课最重要的一张表，**必背**！

| 小调类型 | 上行音阶（A 小调举例） | 下行音阶 | 特点 |
|---|---|---|---|
| **自然 (Natural)** | A B C D E F G A | A G F E D C B A | 与关系大调完全相同的音 |
| **和声 (Harmonic)** | A B C D E F **G♯** A | A **G♯** F E D C B A | 升 7（上下都升） |
| **旋律 (Melodic)** | A B C D E **F♯ G♯** A | A G F E D C B A | 上行升 6 升 7，下行全部还原 |

### 使用频率

| 小调 | 频率 | 主要用途 |
|---|---|---|
| 和声小调 | ⭐⭐⭐⭐⭐（最常见）| 和声进行（V→i） |
| 旋律小调 | ⭐⭐⭐ | 旋律线 |
| 自然小调 | ⭐⭐（少用）| 民谣、摇滚（吉他风）|

### 修改的"焦点"在哪

```
自然小调 → 第 7 音不动
            ↓
和声小调 → 第 7 音升半音 (制造导音)
            ↓
旋律小调 → 第 6、7 音都升半音 (消除增二度)
            （下行时还原）
```

## 9. 实战：A 小调三种版本演奏

### 在键盘上找音

```
钢琴键位（中央 C 区域）：
  ┌─┐ ┌─┐  ┌─┐ ┌─┐ ┌─┐
  │ │ │ │  │ │ │ │ │ │   ← 黑键（升降音）
  └─┘ └─┘  └─┘ └─┘ └─┘
═════════════════════════
  C  D  E  F  G  A  B  C   ← 白键（音名）
```

### 弹奏 A 小调（自然）

```
A → B → C → D → E → F → G → A
全白键，从 A 弹到 A
```

### 弹奏 A 小调（和声）

```
A → B → C → D → E → F → G♯ → A
                          ↑
                      G 旁边的黑键
```

### 弹奏 A 小调（旋律）

```
上行：A → B → C → D → E → F♯ → G♯ → A
                          ↑    ↑
                       F旁黑键 G旁黑键

下行：A → G → F → E → D → C → B → A
       全部回到白键
```

## 10. 练习清单

> 🎯 这一课重点是**听辨 + 弹奏**两手抓，每天 15 分钟。

### 必做（每天）

- [ ] **A 自然小调上下行**（全白键）：右手 1-2-3-1-2-3-4-5 上行，反向下行。
- [ ] **A 和声小调上下行**：注意第 7 音 G♯（按黑键）。
- [ ] **A 旋律小调上下行**：注意上行 F♯ G♯，下行还原 F G。

### 听辨训练

- [ ] **大调 vs 小调**：让人弹一个音阶（不告诉你是大还是小），听出来。
- [ ] **三种小调辨识**：听三个版本的 A 小调，能否区分？

### 关系大小调

- [ ] **5 个对应关系熟记**：
  - C 大 ↔ A 小
  - G 大 ↔ E 小  
  - D 大 ↔ B 小
  - F 大 ↔ D 小
  - B♭ 大 ↔ G 小
- [ ] **正反推导**：给定大调，3 秒说出关系小调；反之亦然。

### 进阶

- [ ] **D 小调三种版本**：比 A 小调多一个降号（B♭）。
- [ ] **听一首小调名曲**：辨识它用的是哪种小调（提示：大多数古典曲用和声小调）。

## 12. 双校准：与通用乐理对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **大调=明亮 / 小调=忧郁** —— 标准音乐心理学共识。
2. **关系大小调共享调号** —— 标准定义。
3. **3 半音法则**（大调下数 3 半音 = 关系小调）—— 正确（即"小三度"关系）。
4. **自然小调是最不常用的** —— 准确（实际演奏中和声/旋律小调更常见）。
5. **导音的牵引力定义** —— 正确。
6. **和声小调升高第 7 音** —— 标准。
7. **旋律小调上行升 6 升 7、下行还原** —— 标准（古典传统旋律小调，**最广泛接受的版本**）。
8. **教会音乐禁止增二度** —— 历史属实（中世纪 Musica Recta 体系）。
9. **和声小调用于和声、旋律小调用于旋律** —— 命名来源准确。
10. **C 小调的关系大调 = E♭ 大调** —— 计算正确。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[00:54] "Equal temperament... is basically having a major scale and a minor scale for each key"** —— Andrew 把"等比律（equal temperament）"和"调式系统"混说了。  
   → **澄清**：
   - **等比律** = 一种**调律方式**（每个半音的频率比都相等）
   - **大小调系统** = 一种**音阶组织方式**
   - 两者**不是同一个东西**，但等比律让所有调都能演奏，所以**支持**了大小调系统的发展。

2. **[02:01] "It's the second lesson, but it doesn't matter"** —— Andrew 自己也记不清是第几课讲的全音半音公式（实际上是第 2 课）。

3. **[10:53] "On the way down... they just lower everything back down"** —— Andrew 说"为什么下行不需要导音了，原因不明"。  
   → **补充**：实际有原因！**下行时音乐"远离主音"，不需要导音的牵引**；下行直到收尾时才会再次升 7。这是 18 世纪音乐家的经验法则。

4. **[15:21] "These ones kind of interdepend"** —— 表达不清。  
   → **澄清**：他想说"和声小调和自然小调相互关联，仅一个音的差别"。

### ❌ 视频中的小错误

1. **[02:32] "Tone tone semitone tone tone tone semitone"** —— 这是**大调**公式，不是小调！Andrew 在解释关系小调时引用，但没明确这是"大调"公式。  
   → **本笔记已澄清**：自然小调的公式是 `T-S-T-T-S-T-T`，与大调不同。

2. **[09:42] Andrew 把"augmented second"和"minor third"在某处混说**：  
   ❌ "They outlawed the interval of a minor third or an augmented second"  
   → **纠正**：被禁止的是**增二度**（如 F→G♯，听起来像两个半音之间的"挤压"），不是小三度（minor third 完全是被允许的，例如 A→C 就是小三度）。增二度和小三度**音响相同**（都是 3 个半音），但**音名不同**，所以书写上是两件事。

3. **[12:08] "Why they call it harmonic minor is because it was used in harmony for chords"** —— ✅ 正确。但**没说全**：
   → **完整原因**：和声小调的第 7 音升高，使得 V 级和弦（建立在第 5 音上）变成**大三和弦**，与主小三和弦形成强烈的"明 → 暗"对比，构成最经典的**完全终止式**。

### 💡 视频遗漏的重要内容

1. **爵士旋律小调（Jazz Melodic Minor）**：现代爵士乐使用的版本是**上下行都升 6、7**（不像古典版下行还原）。这是即兴演奏中重要的音阶。视频未提，但建议后续了解。

2. **多利亚调式（Dorian Mode）vs 自然小调**：多利亚调式 = 自然小调升 6（如 D→E→F→G→A→B→C→D）。在民谣、爵士、电影配乐中**比自然小调更常用**。视频未提。

3. **和声大调（Harmonic Major）**：少见但存在，是大调的"小调化"变体（降第 6 音）。

4. **小调的指法**：A 自然小调的右手指法是 1-2-3-1-2-3-4-5，但和声 / 旋律小调的指法因为黑键的位置而略有差异，视频没讲。

5. **听辨练习方法**：视频鼓励"听辨"但没给具体方法。建议：用钢琴 app（如 EarMaster、Functional Ear Trainer）系统训练。

| 维度 | 评价 |
