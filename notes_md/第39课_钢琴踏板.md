## 2. 钢琴的 3 个踏板

[01:02] **基本介绍**！

### 3 个踏板的位置

```
钢琴底部（从左到右）：
1. 左：Una Corda（"软音 / 弱音"）
2. 中：Sostenuto（"选音"，仅三角钢琴）
3. 右：Sustain / Damper（延音）
        ↓
    用**右脚踩**右踏板（最常用）
    用**左脚**踩左踏板（少见）
            ↓
    脚跟**着地**
    脚趾**踩板**
```

### 立式钢琴的"中踏板差异"

[01:35] **重要！**

```
立式钢琴中踏板：
- 通常是"练习踏板"（Practice Pedal）
- 会**降低音量**（一块毛毡布）
- **不是** Sostenuto！
        ↓
    与三角钢琴**不同**
            ↓
    很多人**误以为**两者相同
```

### Andrew 的"三角钢琴"演示

[01:21] **Andrew 演奏的是三角钢琴**：

```
所以视频中的：
- 中踏板 = Sostenuto（真正的）
- 不是练习踏板
            ↓
    立式钢琴用户注意区别！
```

## 3. 延音踏板（Sustain）— 最重要

[02:00] **本课核心**！

### Andrew 的"95% 法则"

[02:00] *"This is the most common one — 95% of the time we're using the sustain pedal."*

```
延音踏板：
- 钢琴生涯**95%** 时间用它
- 其他两个**几乎不用**
        ↓
    重点学习**这一个**
```

### 工作原理

[02:35] **机械原理**：

```
钢琴内部结构：
- 每个键 → 1 个 Damper（制音器）
- 不弹时 Damper 压住弦
- 弹时 Damper 抬起，弦振动
- 放开 → Damper 落下，止音
            ↓
    踩延音踏板：
    - 所有 Damper 抬起
    - 所有弦**自由**振动
            ↓
    "Damper Pedal" = "解除制音器"
```

### Andrew 的"声音延续"演示

[02:24] **听感**：

```
不踩踏板：
[do]   [立即停]
[do]   [立即停]
        ↓
    像电脑发音
    很**生硬**

踩踏板：
[do]~~~~~~~ （持续到放）
[do]~~~~~~~ 
        ↓
    像歌唱
    很**优美**
            ↓
    钢琴的"灵魂"
```

### 半踩 vs 全踩

[02:44] **关键概念**：

```
全踩（all the way down）：
- Damper 完全抬起
- 共鸣**最大**
        ↓
全放（all the way up）：
- Damper 完全压下
- 共鸣**消失**
            ↓
    中间还有"半踏"
    （高级技术，下文细讲）
```

### 弦的"自由共鸣"

```
踩下踏板时：
- 弹某音 → 该弦振动
- 其他**未弹**的弦也共鸣（Sympathetic Resonance）
- 整个钢琴**同时**响
            ↓
    丰富 + 立体
            ↓
    没有踏板的钢琴 = 没有"魂"
```

## 4. Una Corda（弱音）踏板

[15:08] **左踏板**！

### 名称含义

```
"Una Corda" = 意大利语"一根弦"
        ↓
    历史：早期钢琴每键 1 弦
    踩下后 → 锤子只击 1 弦（弱）
    现代钢琴：每键 3 弦
    踩下后 → 锤子击 2 弦（仍弱）
            ↓
    "Una Corda"是**历史名**
```

### 现代效果

[15:08] **Andrew 的描述**：

```
"This is going to give a much softer, much like sweeter sound."
        ↓
    Una Corda 效果：
    - 音量 **减少**约 30%
    - 音色**更温柔**
    - 像**蒙了一层薄纱**
            ↓
    特别用在：
    - 极弱段落（pp / ppp）
    - 神秘 / 飘渺感
    - 印象派音乐
```

### 立式钢琴的"差异"

[15:55] **重要！**

```
立式钢琴 Una Corda：
- 锤子**整体前移**
- 撞击力变小
- 但**没有**"换弦"效果
        ↓
    效果：仅**减弱音量**
    无**音色变化**
            ↓
    立式比三角"差"
```

### 标记

```
乐谱标记：
- "una corda" = 踩下
- "tre corde" = 放开（"三根弦"）
- "u.c." = una corda 缩写
        ↓
    意大利文术语
            ↓
    所有古典钢琴谱都用
```

### 实战使用

```
最佳应用：
✅ Chopin 夜曲（梦境感）
✅ Debussy《月光》（薄纱感）
✅ Schumann《童年情景》（童真感）
✅ Beethoven《月光》第 1 乐章

❌ 不用：
- 强奏段落
- 节奏驱动段落
- 现代爵士
```

## 5. Sostenuto（中央）踏板

[16:08] **最神秘的踏板**！

### 三角钢琴的功能

[16:13] *"This is going to keep certain notes ringing while you can play other notes that come in and out without holding."*

```
Sostenuto 工作原理：
1. 先按下某些键（不放）
2. **同时**踩下中踏板
3. 放开手指
4. 这些键的 Damper 仍然抬起
5. 之后弹其他键，**正常起落**
        ↓
    "选择性延音"
            ↓
    极少使用！
```

### 实战例子

[18:04] **Andrew 演示**：

```
1. 按下低音 C（保持）
2. 踩中踏板
3. 放开 C（C 仍响）
4. 用同手弹其他音符
5. 其他音符**正常**起落
6. 但 C 一直在响
        ↓
    像"留下一个长音"
            ↓
    Debussy / Ravel 偶尔用
```

### 为什么 99% 钢琴家不用？

```
原因：
1. **极难**配合（双手 + 双脚）
2. 99% 曲目**不需要**
3. 用延音踏板**也能近似**
4. 学习曲线**陡峭**
        ↓
    20 世纪以前**几乎没人用**
            ↓
    Debussy / Ravel 之后**偶尔**
```

### 立式钢琴的"练习踏板"

[16:38] **Andrew 强调**：

```
"Practice pedal — only people in your house can hear you clanking on the keys."
        ↓
    立式中踏板（练习踏板）：
    - 一块**毛毡布**插入弦与锤子之间
    - 音量减弱**70%**
    - 适合**夜晚练习**
        ↓
    不要混淆这两种"中踏板"！
```

## 6. Ped 与 ❋ 标记

[03:40] **乐谱符号**！

### 标记符号

```
踏板的乐谱标记：

Ped     ← "踩下踏板"
❋       ← "放开踏板"

或者：

———L———  ← 踩下持续这段
       ↑
       这里放
```

### Andrew 的"音乐性"原则

[04:06] **关键提醒**：

```
"You don't always have to follow exactly the markings of the composer."
        ↓
    踏板标记是**建议**
    不是**死规则**
        ↓
    根据：
    - 钢琴音色
    - 房间共鸣
    - 自己听感
            ↓
    **耳朵**才是最终标准
```

### 错误的"机械跟随"

```
❌ 错误做法：
- 看到 Ped 就踩
- 看到 ❋ 就放
- 完全**机械**

✅ 正确做法：
- 听音乐流动
- 根据耳朵调整
- **听到浑浊就放**
            ↓
    "Listen, listen, listen"
```

### Beethoven 的特殊踏板（补充）

```
Beethoven《月光》第 1 乐章：
- 标"持续踏板**整段**"
- 但当时钢琴**共鸣弱**
- 现代钢琴**完全不能**整段踩
            ↓
    必须**频繁切换**
            ↓
    示例：踏板标记**过时**
```

## 7. 踏板的"切换"技巧

[09:40] **整集核心技术**！

### Re-pedal 技术

```
切换踏板的目的：
- **保留**当前音延续
- 同时**清理**旧音的浑浊
        ↓
    技术名：**Re-pedal**（重踏）
    或：**Legato Pedaling**（连音踏板）
```

### 4 步法

[10:08] **Andrew 的"完美时机"**：

```
弹下一个新和弦时：
第 1 步：手指**按下**新音
第 2 步：**几乎同时**抬起脚
第 3 步：脚踩**回去**
第 4 步：手指继续**按住**新音
        ↓
    顺序：
    手指下 → 脚抬起 → 脚踩下
            ↓
    时间窗：约 0.1 秒
            ↓
    需要**精确**协调
```

### Andrew 的"Up-Down"心法

[10:51] **简化记忆**：

```
Up（脚抬）= 旧音"清空"
Down（脚踩）= 新音"延续"
        ↓
    Up 和 Down 之间**几乎瞬间**
            ↓
    新和弦 + Up + Down
    = 一个流畅"换气"
```

### 错误时机

[11:04] **常见错误**：

```
错误 1：脚抬太早
- 旧音**断**了
- 失去 legato

错误 2：脚抬太晚
- 旧音**糟蹋**新音
- 浑浊

错误 3：脚抬太长
- 中间**有空隙**
- 不连贯

错误 4：根本不抬
- 全场**共鸣**
- 极**浑浊**
```

### 训练方法

[11:44] **Andrew 的练习**：

```
慢速训练：
1. 弹和弦 1 → 踩
2. 弹和弦 2 → 抬 + 踩（同时）
3. 弹和弦 3 → 抬 + 踩
4. 弹和弦 4 → 抬 + 踩
        ↓
    每天 5 分钟
    1 周内自然
            ↓
    最终：**自动化**
```

## 8. 浑浊声音（Muddy Sound）的危险

[06:13] **整集警告**！

### 什么是浑浊？

```
浑浊声音 = 多个不和谐的音同时响
        ↓
    例：
    - 弹 C 大三 → 踩
    - 然后弹 G7 → 不抬
    - 两者**重叠**响
            ↓
    极不和谐
            ↓
    像"键盘地震"
```

### Andrew 的"地震"演示

[12:05] **生动**：

```
"Notice how this is getting really really muddy?"
[弹一堆音不抬踏板]
        ↓
    所有音叠**在一起**
    像**雷电交加**
            ↓
    "Earthquake on the piano"
            ↓
    可怕！
```

### 为什么会浑浊？

```
原因：
1. **没切换**踏板
2. 切换**太晚**
3. 切换**太短**（脚抬起<0.1 秒不够）
4. 半踩**位置错误**
        ↓
    解决：仔细听 + 精准切换
```

### 浑浊的"美学价值"

[19:00] **Andrew 的反转**：

```
"Sometimes you actually want a little bit of dirty sound."
        ↓
    现代音乐**故意**浑浊：
    - Debussy（印象派）
    - Scriabin（神秘主义）
    - 现代爵士
        ↓
    "适度"浑浊 = 风格
    "过度"浑浊 = 错误
            ↓
    需要**审美**判断
```

### 古典 vs 浪漫的"踏板用量"

[18:42] **历史**：

```
巴洛克：极少用
古典：少量（Mozart / Haydn 的钢琴**没有**踏板！）
浪漫早：中等
浪漫中（Chopin）：大量
浪漫晚（Liszt）：极多
印象派（Debussy）：泛滥
        ↓
    "踏板史" = 钢琴史
            ↓
    Mozart 的踏板版本是**后人**改的
```

## 9. 半踏板（Half Pedal）

[12:54] **进阶技术**！

### 基本概念

```
全踩：踏板按到底
半踏：踏板**只按一半**
        ↓
    Damper **半接触**弦
            ↓
    部分共鸣 + 部分制音
            ↓
    适度浑浊 + 适度清晰
```

### Andrew 的"4 级踏板"

[13:13] **量化**！

```
Andrew 的踏板分级：
Level 0：完全放开
Level 1：1/4 踩下（轻微延音）
Level 2：半踩（中等延音）
Level 3：3/4 踩下（较多延音）
Level 4：完全踩下（最大延音）
        ↓
    专业钢琴家**全部**用
            ↓
    业余只用 0 和 4
            ↓
    差距巨大
```

### 何时用半踏板？

```
✅ 大量音符快速通过：
- 想要**轻微**共鸣
- 但不想**完全**浑浊
        ↓
✅ 渐弱时：
- 半踩 → 渐放
- 平滑过渡
        ↓
✅ 装饰段落：
- 提供"光晕"感
- 但不掩盖旋律
```

### Andrew 的诚实

[13:35] **承认**：

```
"I'm not going to even attempt to do that on this video."
        ↓
    Andrew 自己**没演**示半踏
    因为太**难**
            ↓
    "高级"技术
            ↓
    需要**至少 3 年**练习
```

### 现代研究（补充）

```
半踏板的"科学"：
- Damper **接触面积**变化
- 制音 vs 共鸣 **比例变化**
- 钢琴家用**身体**感受
        ↓
    无法**完全**量化
            ↓
    "艺术 + 直觉"
```

## 10. "Listen, Listen, Listen"耳朵优先

[09:08] **整集黄金法则**！

### Andrew 反复强调

```
[06:05] "Listen, listen, listen."

[14:30] "Listen, listen, listen."
        ↓

            ↓
    最重要的**一句**
```

### 为什么"听"比"踩"重要？

```
原因：
1. 每架钢琴**音色不同**
2. 每个房间**共鸣不同**
3. 每段乐句**情感不同**
4. 每个时代**风格不同**
        ↓
    所有变量都需要**耳朵**判断
            ↓
    没有"标准答案"
```

### Andrew 的"耳朵培养"建议

[09:08] **训练方法**：

```
1. 弹 → 立即**听**
2. 浑浊？ → **抬**踏板
3. 太干？ → **踩**踏板
4. 听感美？ → **保持**
        ↓
    每弹 1 个音
    立即 1 次评判
            ↓
    1 个月**自动化**
```

### Andrew 的"机械跟谱"批判

[18:42] **关键**：

```
"You can't completely write it out — you would never be able to figure that out from the markings."
        ↓
    乐谱标记**有限**
    实际演奏**无限**
            ↓
    必须靠**耳朵**填补
            ↓
    机械跟谱 = 死板
```

## 11. 延伸：作曲中使用踏板

[20:01] **作曲家视角**！

### Andrew 的"作曲范例"

[21:08] **演示**：

```
Andrew 现场即兴一段：
- 用**大量**踏板
- 创造"梦境感"
        ↓
    踏板 = 作曲家"色板"
            ↓
    没有踏板的曲子 = 黑白
    用踏板 = 彩色
```

### 现代作曲家的踏板"魔法"

```
Debussy《月光》：
- 几乎全程踏板
- 制造"月光下"感

Liszt《钟》：
- 重音 + 踏板
- 制造"鸣钟"效果

Rachmaninoff 第 2 协奏曲：
- 大量厚和弦 + 踏板
- 制造"俄式忧伤"

Scriabin 后期：
- 持续踏板 + 怪和声
- 制造"神秘主义"
        ↓
    踏板 = 表达**核心**
```

### 自己作曲时

```
建议：
1. 不要每个音都标 Ped
2. 标**段落级**踏板
3. 留空间给演奏者**自由**
        ↓
    专业作曲家做法
            ↓
    标记简洁 + 演奏自由
```
