## 1. 本课概览

| 曲式 | 段落 | 调性 | 终止式 |
|---|---|---|---|
| **对称二部** | A(4) + B(4) | I → V/vi → I | 开放→封闭 |
| **不对称二部** | A(4) + B(8) | I → V/vi → I | 开放→封闭 |
| **圆舞曲式** | A(4) + B(4)+A'(变化) | I → V/vi → I | 开放→封闭 |
| **三部** | A(4) + B(4) + A | I → V/vi → I | **封闭**→开放→封闭 |

> 💡 曲式是音乐的"地图"——告诉你这首曲子怎么"走完一圈"。

## 3. 二部曲式（Binary Form）总览

[01:05] **本课核心**！

### 基本定义

```
"Binary" = 2（二进制就是 0/1）
        ↓
    曲式 = 2 个段落
        - A 段（一开始）
        - B 段（接续）
            ↓
    最常见的曲式
    18 世纪小步舞曲、奏鸣曲第一主题常用
```

### 三种二部曲式

| 类型 | 特征 |
|---|---|
| **对称（Symmetrical）** | A 和 B 长度相同（4 + 4） |
| **不对称（Asymmetrical）** | A 短，B 长（4 + 8） |
| **圆舞曲式（Rounded）** | B 段末尾"绕回"A，A 变化收尾 |

### 共同骨架

```
全部 3 种二部曲式的共同点：
1. A 段在主调（I）
2. A 段结尾用**开放终止**（暗示未完成）
3. B 段转到属调（V）或关系小调（vi）
4. B 段结尾用**封闭终止**（结束全曲）
        ↓
    主调 → 属调 → 主调
    "出门 → 远行 → 回家"
```

## 4. 对称二部曲式（Symmetrical Binary）

[01:55] **最简单的二部**！

### 结构图

```
对称二部：
| A 段 (4 小节) | B 段 (4 小节) |
| 主调 I       | 转到 V 或 vi → 回 I |
| 开放终止     | 封闭终止       |
        ↓
    总长 = 8 小节
    A 和 B 各占一半 → "对称"
```

### Andrew 的"对话"比喻

[02:13] *"Like one person says something, the other person says something — they have the same number of words."*

```
对称 = "公平对话"
- A 段："我说 4 个字"
- B 段："我也说 4 个字"
        ↓
    平等
    没有"权力斗争"
```

### 实战范例

```
C 大调对称二部：

A 段（4 小节）：
小节 1-3：弹 C 大调主题
小节 4：开放终止（如 I → V）
        ↓
    "开了头但没结束"

B 段（4 小节）：
小节 5-7：转到 G 大调（V）或 Am（vi）
小节 8：封闭终止（V7 → I 回 C）
        ↓
    "去远行又回家"
```

### 听感

```
听众体验：
1. "好听的开头"
2. "等等没结束？"
3. "哦换了个调"
4. "好回到原来的样子了"
            ↓
    "出门散步又回家"的感觉
```

### 实际曲目

```
✅ Bach《G 大调小步舞曲》(BWV Anh.114)
✅ Mozart 早期小步舞曲
✅ 大量初学钢琴教材中的"练习曲"
            ↓
    18 世纪儿童钢琴教学曲常见
```

## 5. 不对称二部曲式（Asymmetrical Binary）

[04:13] **B 段更长**！

### Andrew 的"分蛋糕"比喻

[04:21] *"Asymmetrical means one piece bigger than the other — like cutting cake with my brother."*

```
分蛋糕场景：
- 你切，对方选
- 你必须切均匀（否则吃亏）
- 但有时分配**不均匀**（一块大一块小）
        ↓
    A 段 = 小块
    B 段 = 大块
        ↓
    "不对称"
```

### 结构图

```
不对称二部：
| A 段 (4 小节) | B 段 (8 小节) |
| 主调 I       | 转 V/vi → 回 I |
| 开放终止     | 封闭终止       |
        ↓
    总长 = 12 小节
    A 段短，B 段长
```

### 为什么 B 段长？

[05:31] **Andrew 的解释**：

```
B 段需要：
1. 转调到新调
2. 在新调上展开
3. 然后转回主调
4. 最后封闭终止
        ↓
    需要更多空间
    所以 B 段 = 8 小节
```

### Andrew 自己也搞错过

[10:36] *"Embarrassing — I just realized that I remembered this wrong. It was actually 4 bars and then 4 bars."*

```
Andrew 翻出旧课本看到自己作业：
"咦，对称的也是 8 小节？"
"哦不对，是 4+4 = 8"
"是双段重复"
            ↓
    Andrew 自嘲
    "It's been a long time"
```

### 实战范例

```
C 大调不对称二部：

A 段（4 小节）：
小节 1-3：主题
小节 4：开放终止

B 段（8 小节）：
小节 5-8：在 G 调上展开
小节 9-11：从 G 渐渐回到 C
小节 12：封闭终止
        ↓
    给 B 段更多发挥空间
```

## 6. 圆舞曲式（Rounded Binary）

[05:41] **最有趣的二部**！

### 名称解释

```
"Rounded" = "圆形 / 绕回"
        ↓
    B 段尾巴"绕回"A 段
    然后用变化的 A 结束
            ↓
    像"Loop 循环"
```

### 结构图

```
圆舞曲式：
| A 段 (4 小节) | B 段 (4 小节) | A' 段 (4 小节，变化) |
| 主调 I       | 转 V/vi    | 回 I，结尾**变化**          |
| 开放终止     | 引向 A      | 封闭终止                 |
        ↓
    总长 = 12 小节
    A → B → A'
```

### 关键：A' 是变化的 A

[06:55] **重要！**

```
A 段第一次：
- 4 小节
- 结尾**开放**（比如 I → V）

A' 段（B 段后）：
- 同样 4 小节
- 但结尾**改成封闭**（比如 V → I）
            ↓
    主题相同，结尾不同
    "学到教训了，知道怎么完美结束"
```

### Andrew 的"学到教训"比喻

[08:46] *"Like you've learned your lesson now — now you know how to finish the job."*

```
A 第一次：未完成（开放）
B 段：旅行
A' 段：第二次回来 → 知道怎么完美结束
            ↓
    "成长式"循环
```

### 实战范例

```
小步舞曲常用：
- A：开 4 小节
- B：4 小节
- A'：4 小节（结尾变化）
- 总：12 小节
            ↓
    Beethoven 的"G 大调小步舞曲"
```

## 7. 三部曲式（Ternary Form）

[07:01] **完全不同的曲式**！

### 名称解释

```
"Ternary" = "三部"
**与 binary 完全无关**
        ↓
    Andrew 强调：
    不是"扩展二部"
    是另一类曲式
```

### 结构图

```
三部曲式：
| A 段 (4 小节) | B 段 (4 小节) | A 段 (重复) |
| 主调 I       | 转 V/vi      | 主调 I      |
| **封闭终止** | 开放终止     | 封闭终止     |
        ↓
    总长 = 12 小节
    A → B → A（**完全重复**）
```

### 关键：A 段第一次就是封闭！

[08:00] *"They give you a closed cadence right off the bat — feels like you could be finished there."*

```
三部 vs 二部 的最大差别：

三部 A 段第 1 次：**封闭**（V → I）
- 听感"好像结束了"
- 但实际继续到 B
        ↓
    B 段：开放
        ↓
    A 段重复：仍然封闭
            ↓
    "Closed - Open - Closed"
```

### 与圆舞曲式的关键区分

```
圆舞曲式 (Rounded Binary)：
A（开放）→ B → A'（变化封闭）
        ↓
    A 第一次**未结束**（暗示要继续）

三部 (Ternary)：
A（封闭）→ B → A（同样封闭）
        ↓
    A 第一次**已结束**（但还要展开 B）
```

### 实战范例

```
小步舞曲与三重奏（Minuet & Trio）：
A 段（小步舞曲）→ B 段（三重奏）→ A 段（小步舞曲，重复）
            ↓
    经典三部曲式
```

```
著名作品：
- Mozart 小步舞曲 K.1
- Bach《G 大调小步舞曲》
- 几乎所有 ABA 形式的舞曲
```

## 8. 圆舞曲式 vs 三部曲式 关键区分

[08:21] **整集最容易混淆的点**！

### Andrew 的总结

[08:21] *"This is what so many people get confused — well, is it Rounded Binary or Ternary?"*

### 区分方法

```
看 A 段第 1 次的终止式：

✅ 三部曲式：
A 段第 1 次 = **封闭**（已经"完成"）
        ↓
    "本来就能在这结束"

✅ 圆舞曲式：
A 段第 1 次 = **开放**（"未完成"）
        ↓
    "需要回来再封闭"
```

### 总结表

| 比较 | 圆舞曲式 (Rounded Binary) | 三部曲式 (Ternary) |
|---|---|---|
| A 段第 1 次终止 | 开放（V）| **封闭**（I）|
| A 段第 2 次 | 变化（A'）| 完全重复（A）|
| 节奏感 | "递进" | "对称" |
| 长度 | 12 小节 | 12 小节 |
| 标记 | 通常无 | 通常有 **DC al Fine** |

### Andrew 的"不可靠迹象"

[08:55] *"D.C. al Fine is the big telltale, but it's not a guarantee."*

```
DC al Fine 通常 = 三部
但有些三部不写 DC al Fine
            ↓
    必须靠"听耳朵"
    判断 A 段第 1 次是否"封闭"
```

## 9. DC al Fine 标记

[07:18] **乐谱上的"导航"**！

### 含义

```
DC al Fine：
- DC = "Da Capo"（从头开始）
- al Fine = "到 'Fine' 标记处"
        ↓
    意思：跳回到曲首
    弹到"Fine"为止
            ↓
    用于三部 A → B → A 的"重复 A"
```

### 乐谱格式

```
A 段（带 Fine 标记）...
B 段...
末尾写："D.C. al Fine"
        ↓
弹奏顺序：
1. 弹 A 段（看到 Fine 时记住，但继续弹）
2. 弹 B 段（到 D.C. 时回头）
3. 重新弹 A 段
4. 看到 "Fine" 停止
```

### 为什么不"写 2 遍"？

```
原因：
- 节省纸张
- 标准做法
- 古典乐谱的"惯例"
        ↓
    三部曲式必读 DC al Fine
```

### 其他类似标记

```
DS al Coda：
- DS = "Dal Segno"（从 Sign 标记处）
- al Coda = "到 Coda 标记"
        ↓
    比 DC al Fine 更复杂
    用于含 Coda 的曲子
```

### 奏鸣曲式快速总览（补充）

```
奏鸣曲式（Sonata-Allegro）：
1. **呈示部 (Exposition)**：
   - 第一主题（主调）
   - 桥段（转调）
   - 第二主题（V 调）
   - 结尾（V 调）

2. **展开部 (Development)**：
   - 主题在不同调上展开
   - 紧张、远调

3. **再现部 (Recapitulation)**：
   - 第一主题（主调）
   - 桥段（不再转调）
   - 第二主题（**主调**）
   - 结尾（主调）

4. **尾声 (Coda)**（可选）
            ↓
    Mozart / Haydn / Beethoven 第 1 乐章核心
```

### 回旋曲式（补充）

```
Rondo：
A - B - A - C - A - D - A
        ↓
    A = 反复出现的主题
    B, C, D = 不同插段
            ↓
    像"奏鸣曲第 4 乐章"
    Beethoven 钢琴奏鸣曲常用
```

### 主题与变奏（补充）

```
Theme & Variations：
主题 → 变奏 1 → 变奏 2 → ... → 最终变奏
        ↓
    每次变奏改变某个元素：
    - 节奏
    - 调性
    - 织体
    - 装饰
            ↓
    Mozart "小星星变奏曲"
    Beethoven 32 变奏
```

## 13. 双校准：与通用曲式学对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **二部曲式 A-B 结构** —— 标准。
2. **A 段在主调，B 段转 V 或 vi** —— 标准。
3. **开放 vs 封闭终止** —— 与第 27 课呼应。
4. **三种二部曲式分类** —— 标准。
5. **三部曲式 A-B-A** —— 标准。
6. **关键区分：A 段第 1 次终止式** —— 准确。
7. **DC al Fine 标记** —— 准确。
8. **18 世纪舞曲常用这些曲式** —— 准确。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[02:36] "Symmetrical means evened out"** —— 简化。
   → **校准**：严格说，对称 = "结构对称"（A 和 B 同样长），不是"内容相同"。

2. **[05:14] "It goes for 8 bars"** —— Andrew 自己也搞错过。
   → **校准**：不对称二部的 B 段通常**双倍于 A 段**（8 vs 4）。但具体数字可变。

3. **[06:00] "Rounded binary B section leads back to A section"** —— 表达不严谨。
   → **校准**：
   - B 段不是"导向"A 段
