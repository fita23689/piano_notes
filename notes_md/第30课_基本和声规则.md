## 2. Choral Style vs Keyboard Style

[02:38] **两种和声写法**！

### Choral Style（合唱体）

```
特点：
- 4 个声部分开写
- 符干向上 vs 向下区分声部
- 每个音的"声部"清晰
        ↓
    SATB（女高 / 女低 / 男高 / 男低）
        S：符干向上（高音谱号）
        A：符干向下（高音谱号）
        T：符干向上（低音谱号）
        B：符干向下（低音谱号）
```

### Keyboard Style（键盘体）

```
特点：
- 不区分声部
- 高音谱号写右手
- 低音谱号写左手
- "和弦堆叠"
        ↓
    更适合钢琴演奏
    但**看不出**声部走向
```

### 为什么用 Choral Style 学和声？

[02:55] *"You can see all the voices, and checking your voices is a very big part of harmony."*

```
原因：
1. 看清楚每个声部走向
2. 检查规则是否违反
3. 学会"4 声部思维"
        ↓
    像数学的"画图"
    帮助理解结构
```

### 实战意义

```
学和声 → 用 Choral Style
演奏 → 用 Keyboard Style
        ↓
    学完 4 声部规则后
    在键盘上**自动应用**
```

## 3. 声部进行（Voice Leading）核心原则

[03:48] **本课的"灵魂"**！

### 核心原则：**流畅 (Smoothness)**

[04:13] *"Smoothness is important — and a smooth transition will sound basically good."*

```
流畅 = 各声部尽量"少跳跃"
        ↓
    具体方法：
    1. 共同音 → 保留
    2. 相邻音 → 级进（半音 / 全音）
    3. 跳进 → 仅 Bass 声部
        ↓
    "尽可能少动"
```

### 各声部的"性格"

```
4 声部的运动自由度：

低音 (Bass)：
- 可以**自由跳进**
- 5 度、8 度都允许
- 给曲子"力量"

中音 (Tenor / Alto)：
- 应该**级进**
- 偶尔小跳
- 流畅性优先

高音 (Soprano)：
- 通常**级进**
- 偶尔跳进
- 给曲子"线条"
            ↓
    Bass 是"支柱"
    其他是"丝绸"
```

### Andrew 的总结

[04:21] **核心规则**：

```
"Try to get the notes as close as possible."
        ↓
    最少跳跃
    最大流畅
        ↓
    这是**1 万条规则**的总纲
```

### Andrew 的"1 万条规则"幽默

[03:21] *"Music is creative, we could just do whatever we want — but no, they had to crush my dreams with 10,000 rules."*

```
学和声的"震惊"：
- 期待：自由创作
- 现实：1 万条规则
        ↓
    Andrew 自嘲
    但实际：规则**不到 100 条**
        ↓
    其中**最重要**的 5 条本课覆盖
```

## 4. 运动方向规则

[04:46] **不能全声部同向**！

### 核心规则

[04:48] *"Not all voices should ever move in the same direction — they should never go all up or all down."*

```
4 声部，绝对不能：
- 全部向上
- 全部向下
- 全部不动
            ↓
    必须有"对比"
    至少一个声部反向
```

### 为什么？

```
原因：
- 全同向 = 像"无声部"
- 听感"单薄"
- 不像"4 声部"，像"1 个声部加厚"
            ↓
    破坏 4 声部的独立性
```

### 视觉对照

```
错误（全部下行）：
S: ↓↓↓↓
A: ↓↓↓↓
T: ↓↓↓↓
B: ↓↓↓↓
            ↓
    像一坨往下掉的水

正确（混合方向）：
S: ↑↓↑↓
A: ↓↑↓↑（保持）
T: ↑↑↓↓
B: ↑↑↓↑
            ↓
    像 4 个独立旋律
```

### Andrew 的"舞蹈"比喻

[03:18] *"Someone says let's dance, then says wait, you got to be wrapped in a body cast."*

```
学和声的感受：
- 你想跳舞（创作）
- 老师说："先打石膏"（学规则）
- 你："啊？"
        ↓
    学完规则后才能自由
    像"先学语法再写诗"
```

## 5. 共同音（Common Tone）

[07:30] **平稳进行的"魔法"**！

### 什么是共同音？

```
共同音 = 两个和弦中都存在的音
        ↓
    例：
    C 大三：C - E - G
    F 大三：F - A - C
            ↓
    共同音：C
```

### 共同音的"保留"

```
规则：共同音应该**留在同一声部**
        ↓
    例：
    C 在 Soprano → 下个和弦的 C 也在 Soprano
            ↓
    给曲子"锚点"
    增加流畅感
```

### Andrew 的演示

[07:30] **C 大三 → ?**：

```
"我的高音 C 在 C 大三里"
"下个和弦也保留 C"
        ↓
    例：C → F (共同 C)
    例：C → Am (共同 C)
    例：C → Dm (共同没有，需要其他规则)
```

### 共同音的"特权"

[07:42] **重要例外**：

```
当有共同音时：
✅ 其他声部可以**全部同向**
        ↓
    例：
    Soprano: C → C（共同）
    Alto: E → F（上行）
    Tenor: G → A（上行）
    Bass: C → F（上行）
            ↓
    这是允许的！
    因为 Soprano 保留了 C
```

### 为什么？

```
共同音 = "稳定锚"
        ↓
    其他声部的"齐步走"
    被锚点抵消
        ↓
    总体仍然平稳
```

## 6. 反向运动（Contrary Motion）

[07:54] **最优进行**！

### 什么是反向运动？

```
反向运动 = 两个声部**反方向**移动
        ↓
    例：
    Soprano 上行 + Bass 下行
    Soprano 下行 + Bass 上行
            ↓
    最常用：Soprano vs Bass 反向
```

### Andrew 的"打架"比喻

[03:18] **生动**：

```
反向运动 = 两声部"打架"
- 一个上 + 一个下
- 互相对抗
        ↓
    创造"立体"感
```

### 实战

[07:54] **Andrew 的演示**：

```
"我的 Soprano 全部下行"
"我的 Bass 全部上行"
        ↓
    例：
    Soprano: G → F → E（下行）
    Bass:    C → D → E（上行）
        ↓
    完美的反向运动
```

### 为什么反向运动好？

```
原因：
1. 增加"立体感"
2. 自动避免平行 5/8 度（详见第 8 节）
3. 听感"丰富"
            ↓
    所有教科书的"金科玉律"
```

### 实战策略

```
作曲时优先级：
1. Soprano vs Bass：反向 ✅
2. Alto vs Tenor：保持 / 缓慢
3. 共同音：保留
4. 其他声部：填充
            ↓
    Soprano 和 Bass 是"骨架"
    其他是"血肉"
```

## 7. 声部重叠（Overlap）

[12:08] **常见的"小错误"**！

### 什么是声部重叠？

```
声部重叠 = 一个声部"跨越"另一声部的位置
        ↓
    例：
    Tenor 此时是 F
    下一拍 Tenor 跳到 G（高于 Alto 之前的 F）
        ↓
    Tenor 在某拍**高于** Alto
        ↓
    这就是 overlap
```

### 视觉对照

```
正常情况：
Soprano:   E F G
Alto:      C D E
Tenor:     A B C
Bass:      F G C
            ↓
    每个声部"自己的范围"

声部重叠：
Soprano:   E F G
Alto:      C D E
Tenor:     A B → F（跳到高位）
Bass:      F G C
            ↓
    Tenor 此时**高于** Alto 之前的 F
```

### Andrew 的口头解释

[12:08] *"Overlap = a voice jumps past another voice."*

```
例：
- Tenor 在某拍是 D
- Alto 之前是 F
- Tenor 跳到 G（高于之前的 F）
        ↓
    Tenor "越过"了 Alto
    这是 overlap
```

### 为什么避免？

```
原因：
- 声部"独立性"破坏
- 听众听着"乱"
- 4 声部变成"3 + 1"
            ↓
    破坏纹理
```

### Andrew 的"宽容"

[12:43] **现实**：

```
"Sometimes you're forced into overlap."
        ↓
    完全避免**很难**
    专业作品中**也会出现**
        ↓
    规则：尽量避免，必要时接受
```

## 8. 平行五度与平行八度（禁忌）

[13:38] **整集最严的规则**！

### 平行五度（Parallel Fifths）

```
平行五度 = 两声部同时**5 度**移动
        ↓
    例：
    Soprano: C → D
    Tenor:   F → G
        ↓
    两声部一直保持 5 度
    "平行 5 度"
```

### 平行八度（Parallel Octaves）

```
平行八度 = 两声部同时**8 度**移动
        ↓
    例：
    Soprano: C → D
    Bass:    C → D（低 8 度）
        ↓
    两声部一直保持 8 度
    "平行 8 度"
```

### 严格禁止

[13:46] *"You're not allowed for some reason to have two octaves in a row."*

```
规则：**绝对**禁止
        ↓
    在巴洛克 / 古典作品中
    几乎从来不出现
        ↓
    出现 = 学生作业被打 X
```

### 为什么禁止？

```
原因：
1. 5 度 / 8 度听感**强烈相同**
2. 两声部"合并"成一个
3. 失去 4 声部独立性
            ↓
    在 Choral Style 中"灾难"
```

### 听感测试

```
弹平行 8 度：
[C-c] → [D-d] → [E-e]
        ↓
    听感：像一个"加厚"的旋律
    不像"两个声部"

弹反向运动：
[C-c] → [D-B] → [E-A]
        ↓
    听感：明显"两个声部"
```

### 但现代音乐...

[13:54] **重要提醒**！

```
现代音乐（流行 / 摇滚）：
- 大量平行 5 度
- 平行 8 度也常见
        ↓
    "Power Chord"（电吉他）就是 5 度
    完全没问题
            ↓
    禁忌**仅限**于古典 4 声部和声
```

> ⚠️ **校准**：Andrew 没说"现代音乐可以"。本笔记补充。

### 隐性平行（Hidden Parallels）

[14:26] **更难的规则**！

```
隐性平行 = 用不同音名达到 5 度 / 8 度
        ↓
    例：
    Soprano: D → A（4 度）
    Tenor:   E → B（4 度）
        ↓
    虽然不是"完全 5 度"
    但本质相同
        ↓
    也禁止！
```

### Andrew 的"5 度"详细演示

[14:34] **复杂例子**：

```
设：
Soprano: D → E
Tenor:   A → B
        ↓
    分析：
    - D 到 A 是 5 度
    - E 到 B 也是 5 度
    - 两个 5 度"平行"
        ↓
    禁止！
```

## 9. 声部跨度限制

[15:46] **空间规则**！

### 跨度限制

```
规则：
1. Bass 到 Tenor：最多 10 度（含 10）
2. Tenor 到 Alto：最多 8 度
3. Alto 到 Soprano：最多 8 度
        ↓
    超过 = 声部"飘"
    听感不好
```

### 为什么？

```
原因：
1. 太宽 → 中间"空"
2. 听感分散
3. 失去和弦"整体感"
            ↓
    像 4 个人的合唱团
    站太远 = 声音不融合
```

### 视觉对照

```
合理跨度：
Soprano: G4
Alto:    E4   ← 跨 3 度（OK）
Tenor:   C4   ← 跨 3 度（OK）
Bass:    C3   ← 跨 8 度（极限）
            ↓
    紧凑，融合

太宽跨度：
Soprano: G5（高）
Alto:    E4   ← 跨 10 度（太宽！）
Tenor:   C4   ← 跨 3 度
Bass:    C2（极低）
            ↓
    "撕裂"感
```

### Andrew 的演示

[16:00] **范例**：

```
最大允许的范围：
"Bass 到 Soprano：4 个八度（？）"
        ↓
    Andrew 实际数错
    标准是：Bass-Tenor 10 度，其他 8 度
```

## 10. V7 → I 的解决规则

[18:13] **整集最难的部分**！

### V7 和弦的"4 个音"

```
C 大调中：
V7 = G - B - D - F
- 根音：G
- 三度：B（导音！）
- 五度：D
- 七度：F
        ↓
    每个音都有"专属"解决方向
```

### 4 个音的解决规则

[18:39] **Andrew 的详细教学**：

```
1. 根音 G：
   - 可以**任意**走（自由）
   - 通常下降到 C（低 5 度）
   - 或上升到 C（高 4 度）
   - 或保留为 G（C 大三的 5 度）
        ↓
    最自由的音

2. 三度 B（导音！）：
   - 必须**上行半音**到 C
   - 不能下行
   - 不能保留
        ↓
    "导音必上"！
    这是 V7→I 最严的规则

3. 五度 D：
   - 必须**下行**到 C
   - 不能上行
        ↓
    "属音必降"

4. 七度 F：
   - 必须**下行半音**到 E
   - 不能上行
   - 不能保留
        ↓
    "七度必降"！
    第二严的规则
```

### 实战范例

[20:35] **演示**：

```
V7：
Soprano: D
Alto:    F
Tenor:   B
Bass:    G

I（解决后）：
Soprano: C  ← D 下降到 C
Alto:    E  ← F 下降半音到 E
Tenor:   C  ← B 上升半音到 C
Bass:    C  ← G 下降 5 度到 C
        ↓
    每个声部都按规则"解决"
```

### 为什么这些规则？

```
原因：
- 三全音（B + F）必须解决
- B → C：解决导音的"渴望"
- F → E：解决七度的"张力"
- 这是**调性**的核心
            ↓
    违反 = 听感"未完成"
```

### Andrew 的"毒品交易"幽默

[21:08] *"My theory teacher's son said it sounded like she was doing drug deals."*

```
听 Andrew 老师讲解：
"放下这个，让 E 升起，让 toni 这样，做 4-5-7 减..."
        ↓
    儿子："像在毒品交易"
        ↓
    Andrew 老师 Donna 极其熟练
    甚至能**倒着**做和声作业
```

## 11. 双音法则（Doubling Rules）

[23:32] **3 音和弦如何"4 声部"**？

### 问题

```
和弦：3 个音
声部：4 个
        ↓
    需要"重复"一个音
        ↓
    重复哪个？
```

### 一般规则

[23:39] **Andrew 的总结**：

```
主要和弦（I, IV, V）：
- 重复**根音**
        ↓
    例（C 大调 I）：
    C - E - G + 重复 C
    = C-E-G-C
```

```
次要和弦（ii, iii, vi）：
- 重复**三度** 或 **根音**
        ↓
    例（C 大调 vi = Am）：
    A - C - E + 重复 C 或 A
    = A-C-E-C 或 A-C-E-A
```

```
减七和弦（vii°）：
- 重复**三度**
- 永远不重复**根音**或**五度**
        ↓
    例（C 大调 vii° = Bdim）：
    B - D - F + 重复 D
    = B-D-F-D
```

### V7 不能重复音

[09:14] **特殊**：

```
V7 是 4 音和弦：
G - B - D - F
        ↓
    刚好 4 个声部
    不需要重复
        ↓
    （除非省略某音，但本课不涉及）
```

### Andrew 的总结表

```
和弦 | 重复哪个音
-----|----------
I    | 根音
IV   | 根音
V    | 根音
ii   | 三度 / 根音
iii  | 三度 / 根音
vi   | 三度 / 根音
vii° | 三度 ✅（必须）
V7   | 不重复（4 音）
```

### 为什么？

```
原因：
1. 根音重复 → 加强主调感
2. 三度重复 → 保持和弦色彩
3. 五度重复 → 通常空洞
4. 减七的根音重复 → 不稳定
            ↓
    每个规则都有听觉根据
```

### ⚠️ 表达瑕疵 / 易混淆点

1. **[14:26] "Hidden Parallels"** —— Andrew 没明确讲。
   → **校准**：补充：
   - 隐藏 5 度（Hidden 5ths）：用不同音达到 5 度
   - 同样禁止
   - 检查方法：看任何两声部之间的音程，不能在两个相邻和弦都是 5 度

2. **[15:46] "10 度"** —— Andrew 说"between 5 and 10"。
   → **校准**：标准是 Bass-Tenor 不超过 10 度，其他不超过 8 度。Andrew 表达不严谨。

3. **[16:13] "Anything more than that is too far"** —— 简化。
   → **校准**：
   - 紧密排列 (Close): 所有声部 4 度内
   - 开放排列 (Open): 各声部 8 度内
   - 超过 → 不允许
   - 视频未区分两种排列。

4. **[18:39] "5/3 = 7 chord"** —— Andrew 的简化。
   → **校准**：
   - V7 完整记法："5"（罗马数字）+ 阿拉伯数字 7
   - "5/3"实际是另一种记法（数字低音）
   - Andrew 混着用，可能让初学者混乱。

5. **[23:32] "Double the third in vii°"** —— Andrew 说"all circumstances"。
   → **校准**：vii° 通常重复**三度**，但**有时**可重复根音（视具体情况）。Andrew 的"绝对"说法略简化。

6. **没讲"V7 的省略"**：
   → 实战中常省略 V7 的 5 度（D）
   → 视频未涉及。

### ❌ 视频中的小错误

1. **[15:00] Andrew 自己说"我可能写错了"**：
   → 视频里有几处不严格。
   → 不影响主要规则。

2. **[02:38] "Choral style"** —— 拼写。
   → **校准**：标准拼写是 "Chorale Style"（少一个 "e"）。Andrew 用的"Choral"是同义词的另一种拼写，但不准确。

3. **没讲"反向 5 度 / 8 度"** (Anti-parallel)：
   → 一个上一个下达到 5 度 / 8 度
   → 也应避免
   → 视频未提。

4. **没讲"声部交叉" (Voice Crossing)**：
   → 与 overlap 相关
   → 视频混用了术语。
   - Voice Crossing：在**同一拍** Alto 高于 Soprano
   - Overlap：在**相邻拍** 一个声部跨越另一个的位置
   - 两者**不同**！
