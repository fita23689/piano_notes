## 2. 什么是调号（Key Signature）？

[01:00] *"A key signature is basically a group of sharps or flats that regulates the scale."*

### 工作原理

把这个调中**所有要用到的升降号**，在五线谱开头**集体写好**，然后曲子里就**不用每次再写**了。

### 举例：D 大调

```
全音半音公式: T-T-S-T-T-T-S
D-E-F♯-G-A-B-C♯-D（必须有 F♯ 和 C♯ 才符合公式）
```

### 谱面对比

```
不用调号（满谱乱标）：
| ♯F  G  A  ♯C  D | ♯F  G  A  ♯C  D |   ← 升号写一堆，乱

用调号（开头一次性标）：
[♯♯] | F  G  A  C  D | F  G  A  C  D |   ← 谱开头标"两升"，后面 F C 自动是 F♯ C♯
```

[02:38] *"It brings a lot less clutter to the music and makes it more understandable."*

### 调号的重要特性

1. **位置**：在五线谱开头，**谱号之后、拍号之前**。
2. **作用**：本调中所有同名音都**自动**升/降。
3. **持续性**：贯穿整首曲子，除非有"转调"或"临时记号"覆盖。

### 调号 vs 临时记号

| 特性 | 调号 | 临时记号 |
|---|---|---|
| **位置** | 谱开头 | 音符前 |
| **作用范围** | 整首曲子 | 当前小节 |
| **目的** | 设定基本调性 | 临时偏离 |

[02:47] *"When we have to alter the key signature just temporarily, that's called an accidental."*

## 3. 五度循环：一张图看懂所有调

[09:18] **五度循环（Circle of Fifths）**：把所有 12 个大调按"上行五度"摆成一圈。

### 完整图（视觉化）

```
                  C 大调 (无升降)
              0♯ │ │ 0♭
         G        │        F
       1♯         │        1♭
      ┌──         │         ──┐
     D            │            B♭
   2♯             │             2♭
   ┌──            │            ──┐
   A              │              E♭
  3♯              │              3♭
  ──┐             │             ┌──
   E              │              A♭
  4♯              │              4♭
   └──            │            ──┘
     B            │            D♭
    5♯            │            5♭
       └──        │       ──┘
        F♯       │       G♭
        6♯       │       6♭
              ──┐│┌──
                C♯/C♭
              7♯ │ 7♭
```

### 走向规则

- **顺时针**（C → G → D → A...）：每走一步**多 1 个升号**，音高**上行五度**。
- **逆时针**（C → F → B♭ → E♭...）：每走一步**多 1 个降号**，音高**下行五度**（= 上行四度）。

### 为什么叫"五度循环"？

[23:51] *"They're all moving up in fifths."*

```
C → G  (上 5 度)
G → D  (上 5 度)
D → A  (上 5 度)
... 转一圈又回到 C
```

每走一步刚好**上行 5 度**，所以叫"五度循环"。

## 4. 核心口诀：Father Charles Goes Down And Ends Battle

[05:42] **Andrew 反复强调**这是学调号"最重要的口诀"。

### 口诀本体

```
Father Charles Goes Down And Ends Battle
   F      C    G   D   A   E    B
```

### 用途

这 7 个字母是 **升号（♯）出现的顺序**：

```
1 个升号：F♯
2 个升号：F♯, C♯
3 个升号：F♯, C♯, G♯
4 个升号：F♯, C♯, G♯, D♯
5 个升号：F♯, C♯, G♯, D♯, A♯
6 个升号：F♯, C♯, G♯, D♯, A♯, E♯
7 个升号：F♯, C♯, G♯, D♯, A♯, E♯, B♯
```

### 倒着读 = 降号顺序

[07:00] *"You have to learn it backwards."*

```
Battle Ends And Down Goes Charles Father
   B    E    A   D    G     C      F
```

→ 这就是**降号（♭）出现的顺序**：

```
1 个降号：B♭
2 个降号：B♭, E♭
3 个降号：B♭, E♭, A♭
4 个降号：B♭, E♭, A♭, D♭
5 个降号：B♭, E♭, A♭, D♭, G♭
6 个降号：B♭, E♭, A♭, D♭, G♭, C♭
7 个降号：B♭, E♭, A♭, D♭, G♭, C♭, F♭
```

### Andrew 的搞笑联想

[07:23] *"Sky Captain Charles is in an airplane, looking down at the battlefield where his son is. He gets shot down... Father goes down, battle's over."*

→ 想象空中机长 Charles 在飞机上，看到下方的战场（儿子在那），被击落 → "Father goes down, ends battle"。

> 🧠 **你也可以自创助记**！中文版可以用：
> "**福**(F)**车**(C)**哥**(G)**带**(D)**爱**(A)**儿**(E)**奔**(B) 战场" 之类。

### 升号 vs 降号 选哪个口诀？

[13:32] Andrew 总结：
- **F**orward = **F**lats backward / **S**harps **F**orward  
- **B**ackward = **B**lats（哦不，flats 反着读，**B**attle 开头）

> 🎯 **正向读**（F-C-G-D-A-E-B）= **升号**顺序  
> 🎯 **反向读**（B-E-A-D-G-C-F）= **降号**顺序

## 5. 画五度循环的 6 个步骤

> 🎯 5 分钟内画出完整的五度循环！

### 步骤 1：画一个圆，定 4 个锚点

```
        C (12 点钟位置)
   F♯/G♭                C♯/D♭
        (6 点钟位置)
```

**4 个锚点**：
- **顶部 12 点 = C**（无升降）
- **底部 6 点 = F♯ / G♭**（互为等音调）
- **顶部右侧 = C♯**（最多升号 = 7♯）
- **顶部左侧 = C♭**（最多降号 = 7♭）

### 步骤 2：右半圈填升号调（顺时针）

[10:58] 用 "**Father Charles Goes Down And Ends Battle**"：

```
C → F → C → G → D → A → E → B (← 1点钟到5点钟位置)
            ↑   ↑   ↑   ↑   ↑
         1♯  2♯  3♯  4♯  5♯
```

**注意**：F 是 1 个降号的调，所以它跨在 C 的左边（11 点位置）。**升号侧**主要是 G-D-A-E-B-F♯-C♯。

### 步骤 3：左半圈填降号调（逆时针）

[11:36] 用 "**Battle Ends And Down Goes Charles**"：

```
C → F → B♭ → E♭ → A♭ → D♭ → G♭ → C♭
        1♭   2♭   3♭   4♭   5♭   6♭   7♭
```

**注意**：因为 F 已经在前面用过，所以从 C 直接跳到 F（1 个降号）开始。

### 步骤 4：每个调标注调号数量

```
顺时针：0♯ → 1♯ → 2♯ → 3♯ → 4♯ → 5♯ → 6♯ → 7♯
逆时针：0♭ → 1♭ → 2♭ → 3♭ → 4♭ → 5♭ → 6♭ → 7♭
```

### 步骤 5：填入对应的关系小调（圆圈内层）

```
F 大调 (1♭)  对应  D 小调
C 大调 (0)   对应  A 小调
G 大调 (1♯)  对应  E 小调
D 大调 (2♯)  对应  B 小调
... 以此类推
```

→ **关系小调** = 大调向下数 3 个半音（详见第 8 课）。

### 步骤 6：标出等音调（Enharmonic Equivalents）

```
F♯ 大调 (6♯) ≡ G♭ 大调 (6♭)
C♯ 大调 (7♯) ≡ D♭ 大调 (5♭)
C 大调       ≡ B♯ 大调  (理论上)
```

→ 实际只有 **F♯/G♭** 和 **C♯/D♭** 这两对在演奏中常见。

### 完整结构示意图

```
                     C (0)
                  Am
            F (1♭)      G (1♯)
            Dm            Em
       B♭ (2♭)              D (2♯)
       Gm                    Bm
   E♭ (3♭)                      A (3♯)
   Cm                            F♯m
A♭ (4♭)                            E (4♯)
Fm                                  C♯m
   D♭ (5♭)                      B (5♯)
   B♭m                          G♯m
       G♭ (6♭) ─── F♯ (6♯)
       E♭m              D♯m
            C♭ (7♭) ──── C♯ (7♯)
            A♭m            A♯m
```

## 6. 升号顺序与降号顺序

### 升号始终按 F-C-G-D-A-E-B 顺序加

[14:06] *"E major has all those sharps before this as well."*

举例：E 大调（4 个升号）：
```
F♯ → C♯ → G♯ → D♯
```
（按 Father Charles Goes Down 的顺序，停在第 4 个字母）

### 降号始终按 B-E-A-D-G-C-F 顺序加

举例：A♭ 大调（4 个降号）：
```
B♭ → E♭ → A♭ → D♭
```
（按 Battle Ends And Down 的顺序，停在第 4 个字母）

### 五线谱上的位置规律

> 🎯 调号里的升号/降号**位置固定**！永远写在同一个线/间上。

```
高音谱号 ♯ 顺序（视觉位置）：
       上 5 线 = F♯（第一个）
       上 3 线 = C♯
       上 4 线 = G♯
       上 2 间 = D♯
       上 2 线 = A♯
       中 3 间 = E♯
       中 3 线 = B♯
```

### 重要规律

[15:25] *"Whatever your newest flat is, is going to be what your next scale is."*

> 🎯 **降号侧**：最后一个降号的位置 + 1 度 = 下一个调的主音。

举例：
- 1♭ 调的最后一个降号是 **B♭** → 下一个调是 **E♭ 大调**（B♭ 后退到 E♭）
- 2♭ 调（B♭, E♭）→ 下一个是 **A♭ 大调**（最后一个 E♭ 后退到 A♭）

## 7. 关系大小调在五度循环中的位置

> 🎯 每个大调 12 点钟方向**正下方**（圆圈内层）就是它的关系小调。

### 视觉化

```
       外圈大调
         ↓
  C 大调 (0♯)
       │
       ↓
       A 小调 (0♯) ← 内圈小调
```

### 完整对应表

| 大调 | 调号 | 关系小调 |
|---|---|---|
| C 大 | 0 | a 小 |
| G 大 | 1♯ | e 小 |
| D 大 | 2♯ | b 小 |
| A 大 | 3♯ | f♯ 小 |
| E 大 | 4♯ | c♯ 小 |
| B 大 | 5♯ | g♯ 小 |
| F♯ 大 | 6♯ | d♯ 小 |
| C♯ 大 | 7♯ | a♯ 小 |
| F 大 | 1♭ | d 小 |
| B♭ 大 | 2♭ | g 小 |
| E♭ 大 | 3♭ | c 小 |
| A♭ 大 | 4♭ | f 小 |
| D♭ 大 | 5♭ | b♭ 小 |
| G♭ 大 | 6♭ | e♭ 小 |
| C♭ 大 | 7♭ | a♭ 小 |

### Andrew 的"小一度"小窍门

[25:36] *"For minors, they're always one more than whatever this is."*

观察规律：
```
C 大调 → A 小调 （A 比 G 多 1，G 是 0♯ 的下一个升号）
G 大调 → E 小调 （E 比 D 多 1）
D 大调 → B 小调 （B 比 A 多 1）
```

> 🎯 **每个关系小调的字母 = 比对应大调的"五度循环邻居"多 1 度**。这是 Andrew 自己发现的视觉规律。

## 8. 等音调（Enharmonic Keys）

[09:55] **同一个声音、两种写法**的调。

### 三对等音调

| 等音 1 | 等音 2 | 在键盘上听起来 |
|---|---|---|
| **C♯ 大调** (7♯) | **D♭ 大调** (5♭) | 完全相同 |
| **F♯ 大调** (6♯) | **G♭ 大调** (6♭) | 完全相同 |
| **B 大调** (5♯) | **C♭ 大调** (7♭) | 完全相同 |

### 选择哪个？

```
规则：用调号最少的那个（更易看谱）
```

- **C♯ 大调 (7♯)** vs **D♭ 大调 (5♭)** → 选 **D♭** ✅
- **F♯ 大调 (6♯)** vs **G♭ 大调 (6♭)** → 平手，按曲子需要选
- **B 大调 (5♯)** vs **C♭ 大调 (7♭)** → 选 **B** ✅

[10:00] Andrew 把这叫做 **"inharmonic notes and keys"**（和声不和谐的音/调）—— 但**严格说应该是 enharmonic（等音）**！这是视频里的术语错误，本笔记使用正确名称。

## 9. 转调（Modulation）

[21:48] *"If you're in one key and then you decide to move temporarily to another key."*

### 什么是转调？

曲子中途**换到另一个调**。

### 转调的难度

[22:46] *"You can do a modulation, but to make it sound smooth is another matter."*

→ 转调的"硬技术"很简单（直接换调号），但**让它听起来自然**很难。

### 五度循环的实用价值：相邻调最容易转

[20:54] *"F major, C major, G major are all really closely related keys."*

```
C 大调旁边（左右各一格）：F 大调、G 大调
       ↓ 这些是"近亲"
   转调时，听感最自然
```

### 越远的调，转起来越突兀

```
C 大调 → G 大调（隔 1 步）  顺滑
C 大调 → D 大调（隔 2 步）  稍硬
C 大调 → F♯ 大调（隔 6 步）超刺耳，需要技巧
```

### 不同时代的转调倾向

[22:13] *"In older music it's very limited... once you get to the Romantic Era, modulated all the time."*

| 时代 | 转调频率 | 代表 |
|---|---|---|
| 巴洛克 (1600-1750) | 很少 | 巴赫 |
| 古典 (1750-1820) | 中等 | 莫扎特 |
| 浪漫 (1820-1910) | 大量 | 肖邦、李斯特 |
| 现代 | 极多 | 流行歌曲（每段一个调）|

## 10. 快速识别调号的实战技巧

### 看到 N 个升号，怎么找调？

> 🎯 **法则**：升号侧 → **从 C 出发，沿五度循环顺时针走 N 步**。

```
1♯ → C 顺走 1 步 = G 大调
2♯ → C 顺走 2 步 = D 大调
3♯ → C 顺走 3 步 = A 大调
4♯ → C 顺走 4 步 = E 大调
```

### 看到 N 个降号，怎么找调？

> 🎯 **法则 1**：降号侧 → **从 C 出发，沿五度循环逆时针走 N 步**。  
> 🎯 **法则 2**（更快）：**看倒数第二个降号**就是大调主音！

[15:51] *"Look at how many flats does it have... go back one flat."*

```
3♭ (B♭, E♭, A♭) → 倒数第二个是 E♭ → E♭ 大调 ✅
4♭ (B♭, E♭, A♭, D♭) → 倒数第二个是 A♭ → A♭ 大调 ✅
```

⚠️ **例外**：1♭（只有 B♭）→ 必须记住是 **F 大调**（口诀的第一个）。

### 升号侧的"最后一个升号 + 半音"

> 🎯 **法则**：升号调 → **最后一个升号的音 + 半音 = 大调主音**。

```
1♯ (F♯) → F♯ + 半音 = G → G 大调 ✅
3♯ (F♯, C♯, G♯) → G♯ + 半音 = A → A 大调 ✅
4♯ (F♯, C♯, G♯, D♯) → D♯ + 半音 = E → E 大调 ✅
```

## 11. 练习清单

> 🎯 这一课偏理论，但**学会后看谱速度立马翻倍**。每天 10-15 分钟。

### 必做（每天）

- [ ] **背熟口诀**（目标：1 秒说出）：
  - **正向**：Father Charles Goes Down And Ends Battle (F-C-G-D-A-E-B)
  - **反向**：Battle Ends And Down Goes Charles Father (B-E-A-D-G-C-F)
- [ ] **手画五度循环**：从空白纸开始，5 分钟内完整画出（外层大调 + 内层小调 + 升降号数字）。
- [ ] **N 个升号 → 大调名**：随机抽 5 个数字（如 3, 5, 1, 7, 4），快速说出对应升号大调。

### 进阶

- [ ] **倒数第二降号法**：看任意降号调，3 秒内说出大调名。
- [ ] **关系大小调互推**：给定大调（如 E♭ 大）→ 1 秒说出关系小调（c 小）。
- [ ] **找邻居**：给定 C 大调，列出它的"近亲"调（F 大、G 大、a 小、d 小、e 小）。

### 实战

- [ ] **拿一首曲子**：看调号，立刻说出曲子的调（大/小？什么调？）。
- [ ] **转调识别**：找一首流行歌（如周杰伦的《青花瓷》在副歌处转调），听听转到了哪？

### 听觉训练（选做）

- [ ] **听辨升降号数量**：听一段音阶，能否大致判断升降号数量？

### ⚠️ 表达瑕疵 / 易混淆点

1. **[10:03] "That's called inharmonic notes and inharmonic keys"** —— ❌ **术语错误**。  
   → **正确**：是 **enharmonic**（等音 / 同音异名），不是 inharmonic。
   - **enharmonic**（异名同音）：C♯ ≡ D♭，是同一个音 / 调的不同写法。
   - **inharmonic**（非谐波）：物理学术语，指振动中不符合简单整数比的高次谐波。
   → Andrew 在第 7 课关于"sympathetic vibrations"也说错过类似术语，他可能习惯把 enharmonic 误说成 inharmonic。**本笔记一律使用正确术语**。

2. **[24:08] "Or sorry, not down a fifth, you're going down a fourth"** —— Andrew 在视频里**反复纠正自己**。  
   → **澄清**：从 C 逆时针 → F 是**下行 5 度**（也等同于**上行 4 度**）。Andrew 一开始说错为"下行 4 度"，后来改正。

3. **[25:36] "For minors, they're always one more than whatever this is"** —— 表达极不清晰。

4. **[09:50] "Some people overlap the scales because some scales are actually the same as other scales like G♭ major and F♯ minor"** —— ⚠️ 这里 Andrew 又说错了。  
   ❌ **错误**：G♭ 大调 和 F♯ 小调 **不是同一个**！G♭ 大调（6♭）和 F♯ 大调（6♯）才是等音调。F♯ 小调是 A 大调的关系小调。

### 💡 视频遗漏的重要内容

1. **降号侧的对称性**：每个升号位置在五线谱上**位置固定**（如调号第一个♯永远在 F 线上）。视频强调了顺序但没强调"视觉位置"。

2. **小调的五度循环**：Andrew 用了"内圈小调"，但**没特别说明小调本身也按五度循环排列**（a-e-b-f♯-c♯-g♯...）。这是一个非常优雅的对称性。

3. **教会调式（modes）在循环中的位置**：每个调号对应 7 种调式（Ionian, Dorian, Phrygian, Lydian, Mixolydian, Aeolian, Locrian），不只是大调和小调两种。视频未提，但**爵士和现代音乐**重度依赖这点。

5. **数字调号系统**：流行音乐和爵士乐有时用 "2 升 2 降" 这种**数字简化**，而不是写出具体调号。视频未提。
