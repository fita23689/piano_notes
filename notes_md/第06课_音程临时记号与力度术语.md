## 2. 音程（Intervals）：核心概念

[01:13] *"Intervals — second, third, fourth, fifth..."*

### 定义

**音程 = 两个音之间的距离**，按音阶数级数计算（**包括**两端的音）。

### 八种基本音程（用 C 起算）

| 音程 | 起 | 止 | 视觉 |
|---|---|---|---|
| **一度 (Unison)** | C | C | 同一音 |
| **二度 (2nd)** | C | D | 相邻 |
| **三度 (3rd)** | C | E | 跳一个 |
| **四度 (4th)** | C | F | 跳两个 |
| **五度 (5th)** | C | G | 跳三个 |
| **六度 (6th)** | C | A | 跳四个 |
| **七度 (7th)** | C | B | 跳五个 |
| **八度 (Octave / 8th)** | C | C(高) | 跳六个 |

### 数法口诀

> 🎯 **包含两端**：从 C 数到 G = "C-D-E-F-G" = 5 个音 = 5 度。
> 不要数"中间隔了几个"，要数"一共几个台阶"。

## 3. 和声音程 vs 旋律音程

[01:42] *"There's two types of intervals: harmonic and melodic."*

### 两种写法

| 类型 | 演奏方式 | 谱面表现 |
|---|---|---|
| **和声音程 (Harmonic)** | **同时**弹两个音 | 两个音叠在一起（一上一下）|
| **旋律音程 (Melodic)** | **先后**弹两个音 | 两个音横向排列 |

### 视觉对比

```
和声三度（同时弹 C+E）：    旋律三度（先 C 后 E）：
     E                          E
     C                       C
     ↑同时按下                 ↑先 C  ↑后 E
```

### 命名由来

[07:33] **Harmonic**（和声）→ harmony（和谐）→ "几个音同时响"  
**Melodic**（旋律）→ melody（旋律）→ "一个音接一个音"

### 二度的特殊画法

[02:55] 二度（如 C-D）因为太近，**不能直接叠在一起**（会糊在一起），所以画得**略微错开**：

```
   ⚪    ← D（间）
  ⚪    ← C（线）
```

但实际仍是"同时弹"。

## 4. 识别音程的视觉技巧

> 🎯 **核心规律**：偶数度 vs 奇数度，在五线谱上表现完全不同！

### 偶数度（2、4、6、8）：线↔间

[03:45] *"An even interval is always going to be going from a space to a line or a line to a space."*

```
        二度 (2)：相邻的"间→线" 或 "线→间"
        四度 (4)：跳一个，仍是"间↔线"
        六度 (6)：跳两个，仍是"间↔线"
        八度 (8)：跳三个，仍是"间↔线"
```

→ **特征**：音符位置在五线谱上**类型不同**（一个在线，一个在间）。

### 奇数度（3、5、7）：线↔线 或 间↔间

[04:14] *"They always go from one type to another type — same type."*

```
        三度 (3)：相邻的"线→线" 或 "间→间"
        五度 (5)：跳一个，仍是"同类型"
        七度 (7)：跳两个，仍是"同类型"
```

→ **特征**：音符位置在五线谱上**类型相同**（都在线，或都在间）。

### 视觉对比图

```
偶数度（一线一间）：
═══════════
       ⚪
═══════════
   ⚪
═══════════

奇数度（两线 或 两间）：
═══════════
═══════════ ⚪
═══════════
═══════════ ⚪
═══════════
```

### 实战识别流程

```
1. 看两个音的位置类型（都在线？都在间？还是一线一间？）
   ├─ 一线一间 → 偶数度（2/4/6/8）
   └─ 同类型 → 奇数度（3/5/7）

2. 看跨度
   ├─ 紧挨着 → 2 或 3
   ├─ 跳一格 → 4 或 5
   ├─ 跳两格 → 6 或 7
   └─ 跳三格 → 8（高八度）
```

## 5. 音程式读谱（Intervalic Reading）

[04:54] *"Learning to recognize intervals quickly is extremely important to reading music quickly."*

### 为什么要学这个？

❌ **慢的读谱方式**：逐音念名 "G、F、A、B、C、D、C、B、A、G"...（手忙脚乱）  
✅ **快的读谱方式**：读"上下方向 + 度数" "上 2、下 3、上 4、下 3、上 6"...（手指自动移动）

### Andrew 的实战建议

[05:12] *"You can just read it like that and tell your hand to do those specific instructions."*

### 训练方法

[08:50] *"Get cards, flash cards, and look at those intervals very, very closely. Do those intervals like mad crazy."*

→ **音程闪卡 2 周** = 读谱速度质变。

| 训练阶段 | 时长 |
|---|---|
| 第 1 周 | 每天 5 分钟，识别简单的 2、3、5 度 |
| 第 2 周 | 加入 4、6、7、8 度，速度提到 2 秒/卡 |
| 之后 | 不再刻意训练，自然就会 |

### Andrew 的极力推荐

[09:08] *"I highly highly highly highly recommend that you do flashcards for intervals."*  
（连说四个 highly！）

## 6. 临时记号（Accidentals）

[10:11] *"They're called accidentals."*

### 三种符号在五线谱上的写法

| 符号 | 名称 | 看起来像 | 效果 |
|---|---|---|---|
| **♯** | 升号 (Sharp) | 井号 # | 升高半音 |
| **♭** | 降号 (Flat) | 小写 b | 降低半音 |
| **♮** | 还原号 (Natural) | 小方框 | 取消升降 |

### 重要规则 1：写在音符之**前**

[11:26] *"You always have to put them before the note."*

```
✅ 正确：♯ ⚫    （先看到升号，再弹音）
❌ 错误：⚫ ♯    （已经弹了，再加升号没用）
```

### 重要规则 2：作用范围 = 当前小节

[10:31] *"Accidentals are in effect until the end of the bar unless there's something else."*

```
小节 1：| C♯, D, C, ?, ? |
                       ↑
              这里没写升号，但因为前面有 C♯，
              这两个 C 都自动是 C♯（除非写 ♮）
```

到下一小节自动失效：

```
小节 1：| C♯ ... | 小节 2：| C ... |
            ↑                   ↑
       这里是 C♯              这里是 C（自动还原）
```

### Andrew 的"监狱"比喻

[10:55] *"It's like a criminal that can't escape jail, but it can wander around in its cell unless you shoot it dead."*  
→ 临时升降号像"被关在小节牢里"，关到本小节结束就放出来；除非"开枪"（用 ♮ 还原）。

### ⚠️ 初学最容易踩的坑

```
| C♯ → C → C |    错：弹"升 C，C，C"
                    对：弹"升 C，升 C，升 C"（除非有 ♮）
```

→ **一定记住**：临时记号在小节内**自动持续生效**！

## 7. 力度术语：四个基础层级

[12:24] 音乐里的力度用**意大利语**表示（因为西方音乐的中心是 18 世纪的维也纳，主要语言是意大利语）。

### 四个基础符号

| 符号 | 全称 | 意大利语原意 | 中文 |
|---|---|---|---|
| **f** | forte | "强" | 大声 |
| **p** | piano | "柔" | 小声 |
| **mf** | mezzo-forte | "中等强" | 中等大声 |
| **mp** | mezzo-piano | "中等柔" | 中等小声 |

### 力度阶梯（从弱到强）

```
最弱   pp < p < mp < mf < f < ff   最强
       ↑              ↑
       双 p（很弱）    双 f（很强）
```

[16:24] 双字母 = 加强版：
- **pp** (pianissimo) = 很弱
- **ff** (fortissimo) = 很强

更极端的还有 ppp、fff（少见）。

### 记忆口诀

| 符号 | 联想 | 助记法 |
|---|---|---|
| **f** | fireworks 烟花 | 烟花 = 大声 |
| **p** | pillow 枕头 | 枕头 = 小声 |
| **m** | moderate 适度 | "中等" |

[15:36] *"Think of fireworks for f. Think of pillow for p — like I'm going to hit the pillow and go to sleep."*

### 钢琴名字的由来

[14:24] **Pianoforte（钢琴的全名）= piano + forte = "弱+强"**！

> 🎯 钢琴最早叫 **fortepiano**（强弱琴），因为它是第一个**能控制力度**的键盘乐器（之前的羽管键琴 harpsichord 力度固定）。

[15:01] *"This thing that we're playing — its real full name would be a 'loud-soft.'"*

## 8. 渐强与渐弱

[18:50] 这两个符号是"音乐版的箭头"。

### 渐强（Crescendo）

```
        ────────
       <        音量越来越大
        ────────
```

| 符号 | 含义 |
|---|---|
| **<** 或越张越开的两条线 | 渐强 |
| **cresc.** | 缩写 |
| **crescendo** | 全称 |

### 渐弱（Decrescendo / Diminuendo）

```
        ────────
                >  音量越来越小
        ────────
```

| 符号 | 含义 |
|---|---|
| **>** 或越收越窄的两条线 | 渐弱 |
| **decresc.** 或 **dim.** | 缩写 |
| **decrescendo** 或 **diminuendo** | 全称 |

### 视觉记忆

> 💡 想成"**音量喇叭**"：开口大的一头 = 大声；开口小的一头 = 小声。开口由小变大 = 渐强；由大变小 = 渐弱。

### 实战举例

[19:48] Andrew 演奏示范：从 **p** 渐强到 **f**，再从 **f** 渐弱到 **p** —— 像浪潮一样有起伏，曲子才有"呼吸感"。

## 9. 断音（Staccato）vs 连音（Legato）

[20:43] *"There's two types of touches when you're playing piano."*

### 断音（Staccato）

| 标记 | 演奏方式 | 听感 |
|---|---|---|
| **音符上方/下方一个小点 ⚫·** | 短促，立刻松开 | "蹦-蹦-蹦"，像跳跃 |

[21:23] *"Think that the keys are hot — they're like 'ow!' You don't want to hold on to the keys very long."*

### 触键比喻

> 🔥 **想象琴键烫手**：碰一下立刻缩回。这就是 staccato 的感觉。

### ⚠️ 不要和附点混淆

```
staccato：⚫·       (点在音符上方/下方)
dotted：  ⚫.       (点在音符右边)
```

→ 位置不同，意思完全不同！

### 连音（Legato）

| 标记 | 演奏方式 | 听感 |
|---|---|---|
| **一条弧线 ⌒ 把多个音连起来** | 流畅，音与音之间无间隙 | "啦~啦~啦~"，像歌唱 |

[22:11] *"Legato — that's smooth. Joined together."*

### 触键比喻

> 🌊 **想象在水里游泳**：手指像在水中划过，每个音流畅过渡到下一个。

### 演奏原则

```
没有标记时 → 默认连奏（legato）
有 ⚫· 标记 → 断奏（staccato）
有 ⌒ 标记 → 强调连奏
```

[24:00] *"If you don't see either one, you can just assume that you'll probably want to play it legato."*

### 乐句（Phrase）= 大连音线

[24:18] **乐句标记**就是"大版本的连音线"，可能跨越很多音符。

> ☁️ **比喻**：把乐句想成**云朵**，云朵里的音都要连起来；云朵之间是**蓝天**，需要"抬手呼吸"。

[24:36] *"You always lift at the end of phrase marks. Be very graceful when you do your lifts."*

## 10. 连音线 vs 延音线（Slur vs Tie）

[25:11] **看起来一模一样，但意义完全不同！**

### 视觉对比

```
连音线 (Slur)：       ⌒
                  ⚫       ⚫     （不同的两个音）
                  C         D

延音线 (Tie)：        ⌒
                  ⚫       ⚫     （相同的两个音）
                  C         C
```

### 关键区别

| 项 | 连音线 (Slur) | 延音线 (Tie) |
|---|---|---|
| **连接的音** | **不同**音名 | **相同**音名 |
| **演奏方式** | 两个音都弹，但要平滑过渡 | **只弹第一个**！第二个加进去成为延长 |
| **作用** | 表达连贯性 | 延长时值 |

### 延音线的时值合并

[26:02] *"This note and this note are actually going to become one note and you're going to hold it all the way to the end."*

```
四分音符 ⌒ 四分音符  =  二分音符 (1+1=2 拍)
四分音符 ⌒ 二分音符  =  附点二分音符 (1+2=3 拍)
全音符 ⌒ 全音符      =  双倍全音符 (4+4=8 拍)
```

### 为什么要用延音线？

延音线常用于**跨小节延长**：

```
| ... ⚫ ⌒ | ⚫ ... |
       ↑      ↑
       小节末   小节首
       
（两个 C 用延音线连起来，相当于"一个跨小节的长音"）
```

→ 因为五线谱有"小节线"必须切断，但音乐想"延续"，所以用延音线"缝合"过去。

### 区分要点

> 🎯 **看音名**！如果两个音是**同一个音**（C 接 C，G 接 G）→ Tie。如果**不同**（C 接 D，E 接 G）→ Slur。

[26:23] *"Slurs always go from like an F to a G — they go to a different note. A tied note has to stay the same."*

## 11. 练习清单

> 🎯 这一课全是识谱杂项，每天 10 分钟巩固。

### 必做（每天）

- [ ] **音程闪卡**：4 张/天，识别 2、3、5、8 度（和声音程）。
- [ ] **背 4 个力度**：f / p / mf / mp 默写中文意思。
- [ ] **临时记号小测**：随手画 5 个 ♯/♭/♮，标到正确位置（音符前 + 同一线/间）。
- [ ] **小节范围测试**：写一段虚构乐句，标出"哪些音受前面 ♯ 影响"。

### 进阶

- [ ] **音程闪卡升级**：所有 8 种音程，2 秒内识别。
- [ ] **听辨 staccato vs legato**：找一段巴赫的赋格 vs 莫扎特奏鸣曲，听出对比。
- [ ] **延音线找茬**：拿一首流行歌谱子，圈出所有 tie 和 slur，确认你能区分。

### 实战

- [ ] **跟着力度演奏**：找一首带 p/f/cresc 的简单曲子，按标记练习力度变化。
- [ ] **意大利语单词记 5 个**：forte、piano、mezzo、crescendo、staccato。

1. **[02:21] "Free flowing whatever crap"** —— Andrew 自嘲式语言，但也透露视频例子**没拍号**。  
   → **澄清**：实际乐谱必须有拍号，本课是为了演示"音程"才省略。

2. **[06:18] "Whether it'll be a major third or minor third is a different question."** Andrew 提到了大三度/小三度但没展开。  
   → **补充**：
   - **大三度 (Major 3rd)**：4 个半音（如 C-E）
   - **小三度 (minor 3rd)**：3 个半音（如 C-E♭）
   - 大调和弦用大三度+小三度叠加；小调和弦反过来。
   → 这是**音程"质"**的概念（major/minor/perfect/augmented/diminished），本课只讲了"度数"（quantity），没讲"质量"。

3. **[16:32] "MF is moderately loud, MP is moderately soft"** —— 准确。但 **mp 实际比 p 略响** 这个细节容易反直觉。  
   → **澄清**：力度阶梯精确为 `pp < p < mp < mf < f < ff`，注意 **mp 比 p 响**（moderate piano 是"半收着"的弱）。

4. **[21:42] "A staccato doesn't lengthen the value"** —— 严格说 staccato 是**缩短**音的实际响时（约一半时值），不是不改变。  
   → **澄清**：
   - 原本的四分音符占 1 拍
   - 加 staccato 后：声音只响约 0.5 拍，剩下 0.5 拍是静音
   - 但**节奏位置不变**：下一个音仍在原位置进入

5. **[16:11] "It's 3:00 in the morning"** Andrew 多次提到自己凌晨录视频累，部分句子表达跳跃。  
   → 本笔记已结构化重整。

### ❌ 视频中的小错误

1. **[10:31] "C# until the end of the bar and then we'd go back to plain C because it gets cancelled at the end of the bar."** —— ✅ 对的，但他**没强调一个例外**：**用延音线 (tie) 跨过小节线**时，前面的临时记号**仍然有效**到延音线结束。
   → **补充**：`| C♯ ⌒ | C ... |` 这个跨小节的 C 仍然是 C♯。

### 💡 视频遗漏的重要内容

1. **音程的"质"** —— Andrew 只讲了度数（2/3/4/5...），没讲质（major/minor/perfect/augmented/diminished）。这是后续学习和声、调性的基础，但视频未涉及。
   - 后续可学：`Perfect 4th / 5th / 8ve（完全四五八度）` `Major / minor 2nd / 3rd / 6th / 7th`

2. **同度（Unison）和复音程**：视频从二度起步，没讲"同度（一度）"和大于八度的"九度、十度..."。

3. **力度词的更高级形式**：sf（突强）、sfz（极突强）、fp（强后立刻弱）等。本课未提，初学不必管。

4. **断音的细分**：staccato (.) 还有更短的 staccatissimo（▼，超短促）和 mezzo-staccato（半断半连）等变体。

5. **跨小节 tie 时临时记号的延续** —— 见 ❌ 第 1 点。
