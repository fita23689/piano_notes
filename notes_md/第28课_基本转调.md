## 2. 转调（Modulation）的定义

[00:13] *"Tonalization, modulation, applied dominant chords."*

### 核心定义

```
转调 (Modulation) = 从一个调"改变"到另一个调
        ↓
    例：C 大调 → G 大调
        C 大调 → F 大调
        C 大调 → A 小调
            ↓
    一段时间后**确立新调**
```

### 为什么要转调？

```
1. 增加变化：避免单调
2. 表达情感：不同调有不同情绪
3. 戏剧效果：突然转调 = 突然变化
4. 结构需要：奏鸣曲式必须转调
            ↓
    没转调的曲子 = 单调
    有转调的曲子 = 立体
```

### 调性化 vs 转调

```
调性化 (Tonalization)：
- **短期**进入新调（2-4 拍）
- 然后回原调
- "蜻蜓点水"

转调 (Modulation)：
- **长期**进入新调（数小节以上）
- 在新调上展开
- "正式入住"
            ↓
    时长决定术语
```

> 💡 **本课重点**：**调性化**比真正的"转调"更常见。

## 3. 应用属和弦（Applied Dominant）

[01:00] **最常用的转调技术**！

### 什么是应用属和弦？

```
应用属和弦 = "目标和弦"的属和弦
        ↓
    不是当前调的 V，是**临时**的 V
            ↓
    例（C 大调）：
    要去 G？ → G 的 V 是 D
    所以用 D 大三和弦"指向"G
            ↓
    D 是"应用属和弦"
    D 是 V/V（V of V）
```

### 原理

```
任何和弦都可以"被它的属和弦引出"
        ↓
    像"红地毯"：
    - 应用属和弦 = 红地毯
    - 目标和弦 = VIP
    - 听众感觉"VIP 即将到达"
            ↓
    创造"转调感"
```

### 罗马数字标记

```
V/V = "five of five"
意思：当前调的 V 的 V
        ↓
    例（C 大调）：
    V = G 大三
    V/V = D 大三（G 的 V）
            ↓
    其他例子：
    V/IV = C 大三（F 的 V）
    V/vi = E 大三（Am 的 V）
    V/ii = A 大三（Dm 的 V）
```

### 实战例子（V/V → V → I）

```
C 大调中：
1. I (C-E-G)
2. V/V (D-F♯-A)
3. V (G-B-D)
4. I (C-E-G)
        ↓
    F♯ 是关键（C 大调没 F♯）
    F♯ 出现 → 听众"咦？"
    然后 F♯ → G → C → 解决
            ↓
    先制造"惊讶"
    再"解决"
    完美
```

## 4. V/V 的演奏与听感

[03:18] **听觉测试**！

### Andrew 的演奏

```
慢速演奏：
[C E G C] 一拍
[D F♯ A D] 一拍
[G B D G] 一拍
[C E G C] 一拍
        ↓
    四个"清晰的"步骤
```

### 听感分析

```
第 1 拍：稳定（在 C）
第 2 拍：意外（F♯ 不在 C 调）
第 3 拍：缓解（G 是熟悉的属）
第 4 拍：圆满（回 C）
        ↓
    "意外 → 缓解 → 圆满"
    制造小型戏剧
```

### Andrew 的"完美终止套娃"

[03:58] *"That's a perfect cadence, and this is also a perfect cadence."*

```
V/V → V：
- D → G 是 G 调中的 V → I
- 微型完全终止

V → I：
- G → C 是 C 调中的 V → I
- 真正的完全终止
            ↓
    两个嵌套的完全终止
    像"俄罗斯套娃"
```

### Andrew 的"shock"评价

[06:12] *"The applied dominant is sort of like a shock — like throwing someone from hot tub into cool pool."*

```
应用属和弦的特点：
✅ 直接、有冲击
✅ 听感"突然"
✅ 情绪转折鲜明
❌ 不太流畅
            ↓
    用于"戏剧性"段落
```

## 5. 调性化（Tonalization）vs 转调

[02:42] **重要术语区分**！

### Andrew 的定义

```
调性化 (Tonalization)：
- 极短（2 个和弦左右）
- "暂时"指向新调
- 立即回原调
        ↓
转调 (Modulation)：
- 较长（几小节）
- "真正"进入新调
- 在新调上展开
            ↓
    都是"换调"，长短不同
```

### 时间标准（Andrew 的）

```
调性化：< 2 小节
模糊地带：2-4 小节
转调：> 4 小节
            ↓
    无严格分界
```

### 听众的感受

```
调性化听感：
- "咦，刚才离开了 C 调一会儿"
- "现在又回来了"

转调听感：
- "好像换调了"
- "新调持续了一段时间"
- "现在在新调"
            ↓
    时长决定听众"察觉"程度
```

### 实战应用

```
调性化用法：
- 强调某和弦
- 短暂戏剧
- 经过性变化

转调用法：
- 主题展开
- 新段落
- 戏剧高潮
            ↓
    选哪个看曲式需要
```

## 6. 共同和弦（Pivot Chord）

[06:43] **平滑转调技术**！

### 定义

```
共同和弦 = 在两个调中**都存在**的和弦
        ↓
    用作"桥梁"
    从一个调"溜进"另一个调
        ↓
    听众**感觉不到**转调发生
```

### 实战例子

```
要从 C 大调转到 F 大调：

找共同和弦：
- F 和弦在 C 中是 IV
- F 和弦在 F 中是 I
        ↓
    F 是共同和弦！

完整流程：
1. C: I → vi → iii → IV (F)
   ↑此时 F 是 C 中的 IV
2. F: I → V → I
   ↑此时同样的 F 变成 F 中的 I
            ↓
    转换悄无声息
```

### Andrew 的演示流程

[07:00] **完整范例**：

```
C 大调段：
- I (C major)
- vi (Am)
- iii (Em)
- IV (F major) ← 共同和弦！

转调到 F 大调：
- 同一个 F major 现在是 I
- V (C major 7) ← 加了 B♭！
- I (F major)
            ↓
    B♭ 出现 → 听众"哦，确实换调了"
```

### "潜伏"的转调

[08:00] *"Your ear doesn't actually know that you're in a different key until you hear another key that throws you off."*

```
关键洞察：
- F 和弦本身**听不出**调性
- 必须有"调外音"才确认换调
- F 调的标志：B♭
- C 调的标志：B 还原
            ↓
    一个 B♭ → 调性确立
    一个 B 还原 → 调性归回
```

### 实战意义

```
共同和弦转调：
- 听众"事后"察觉
- 没有"突兀"感
- 优雅 / 流畅
            ↓
    巴赫 / 莫扎特最爱
    主题展开常用
```

## 7. 转调的"潜伏"听觉

[08:00] **本课最深刻的听觉规律**！

### 核心规律

```
"听众察觉转调"的时刻 ≠ "实际转调"的时刻
        ↓
    实际转调：你写下"换调"的瞬间
    听众察觉：第一个**调外音**出现的瞬间
            ↓
    两者可能差几拍 / 几小节
```

### Andrew 的"魔术"理解

[10:23] *"Technically we're in F major here, but our ear still thinks we're in C major. Then we realize we're in F major here. But by the time we get to here, we're back in C major, but our ear doesn't hear it until here."*

```
作曲家的视角：
位置 1：我决定转 F → 实际换调
位置 2：F 调持续
位置 3：我决定回 C → 实际换调
位置 4：C 调持续

听众的视角：
位置 1：还在 C
位置 1.5：B♭ 出现 → "换调了！"
位置 2：在 F
位置 3：还在 F（认为）
位置 3.5：B 还原 → "回 C 了！"
位置 4：在 C
            ↓
    听众**永远滞后**于作曲家
    这是转调的"魔术"
```

### 为什么这"很 cool"？

```
作曲家利用此规律：
1. 提前转调（听众不察觉）
2. 突然引入调外音（听众惊讶）
3. 在新调展开（听众适应）
4. 提前回调（听众不察觉）
            ↓
    转调有"潜伏期"
```

## 8. 2-7 转调

[13:02] **最强的转调技术**！

### 什么是 2-7 转调？

```
2-7 转调 = 用 ii7 → V7 → I 序列转入新调
        ↓
    其中 ii7 和 V7 是**新调**的
        ↓
    例：转 D 大调
    1. 用 D 调的 ii7（Em7）
    2. 接 D 调的 V7（A7）
    3. 解决到 D 调的 I（D 大）
            ↓
    "强力"建立新调
```

### Andrew 的演示

[15:00] **C → D 大调**：

```
完整流程：
1. C 大调 I (C-E-G)
2. C 大调 IV6/4 (F-A-C 转位)
3. D 大调 ii4/3 (Em7 转位) ← 2-7 开始
4. D 大调 V7 (A7)
5. D 大调 I (D-F♯-A) ← 落地新调
            ↓
    "强力锁定"D 大调
```

### 为什么用 ii7？

```
ii7 → V7 → I 是**爵士最常用**进行
        ↓
    ii7 = "前驱"
    V7 = "属"
    I = "主"
            ↓
    强力"指向"新调
```

### 远调转调的"工具"

[14:14] *"You can go to far off weird keys very easily."*

```
近调（C 调中近调：F、G、Am）：
- 共同和弦容易找
- 共同和弦转调即可

远调（C 调中远调：D、E、A、B、F♯）：
- 共同和弦少
- 必须用 2-7 转调"强行"建立
            ↓
    2-7 是"远调神器"
```

### Andrew 的"二次应用属"

[13:39] *"The 2-7 modulation is sort of an applied dominant of an applied dominant."*

```
分析：
- ii7 → V7 是 V7 的 V/V
- V7 → I 是 I 的 V
            ↓
    嵌套两层"指向"
    比单层 V/V **更强**
```

## 9. 近关系调 vs 远关系调

[14:31] **五度循环视角**！

### 近关系调

```
C 大调的近关系调：
- G 大调（5 度上）
- F 大调（5 度下）
- Am 小调（关系小调）
- Em 小调（5 度上的关系小调）
- Dm 小调（5 度下的关系小调）
            ↓
    共有 5 个调
    转过去**容易**
```

### 远关系调

```
C 大调的远关系调：
- D 大调（2 度上）
- A 大调（3 度上）
- E 大调（更远）
- B 大调（远）
- F♯ 大调（极远）
- D♭ 大调（远）
- G♭ 大调（极远）
            ↓
    共有 8+ 个调
    转过去**难**
```

### Andrew 的五度圈视图

[14:51] **画图说明**：

```
五度圈（顺时针 = 升号增加）：
C → G → D → A → E → B → F♯/G♭ → ...

C 的"近邻"：
- G（顺时针 1 步）
- F（逆时针 1 步）
            ↓
    转过去 = "走 1 步"
    转远调 = "走多步"
```

### 转调难度排序

```
1. 同主音转调（C → Cm）：极易
2. 关系大小调转调（C → Am）：易
3. 五度近邻（C → G / F）：易
4. 三度关系（C → Em / E）：中
5. 二度关系（C → D）：难
6. 三全音关系（C → F♯）：极难
            ↓
    远 = 用 2-7 转调
```

   → C 大 → A 小 是最常见
