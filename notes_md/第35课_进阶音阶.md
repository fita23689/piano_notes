## 3. 三度音阶（Scales in Thirds）

[01:14] **本课核心**！

### 基本定义

```
普通音阶：单音，从 C 到 C
三度音阶：双音，每对相距 3 度
        ↓
    例（C 大调三度音阶）：
    [C+E] - [D+F] - [E+G] - [F+A] - [G+B] - [A+C] - [B+D] - [C+E]
        ↓
    8 对双音
    每对都是"三度"
```

### 视觉对照

```
普通音阶：
C - D - E - F - G - A - B - C
        ↓
    8 个单音

三度音阶：
[C-E] - [D-F] - [E-G] - [F-A] - ...
        ↓
    每对相隔 3 度
    像"双音爬梯"
```

### 关键技术：起点偏移

[01:50] **重要细节**：

```
左手起点：C
右手起点：**E**（不是 C！）
        ↓
    右手"提前 3 度"
            ↓
    左手到 C8（最高音）时
    右手到达 E8
        ↓
    左手停 → 右手再走 2 步
```

### 指法（C 大调）

```
左手三度音阶（5-3-2-1-3-2-1-3）：
[C+E]：5+3
[D+F]：4+2
[E+G]：3+1
[F+A]：2+1（拇指穿过）
[G+B]：1+3
...

右手三度音阶（1-3-1-2-1-3-1-2）：
[E+C]：3+1
[F+D]：4+2
[G+E]：1+3
[A+F]：2+1
...
```

### 演奏要点

[02:43] **Andrew 强调**：

```
"Try to make both your hands go in a straight line — level — no bumps."
        ↓
    关键：
    1. 双手"齐步走"
    2. 没有"抖动"
    3. 力度均匀
        ↓
    像"两人三足"
```

### Andrew 的"拇指穿过"提醒

[03:01] **难点**：

```
"Your hands are going to be doing tucks at different times."
        ↓
    左右手的"拇指穿过"位置**不同步**
    左手在某拍过指
    右手在另一拍过指
            ↓
    必须**独立练**习
            ↓
    一开始很别扭
    熟练后就自然了
```

## 4. 六度音阶（Scales in Sixths）

[01:20] **视频未深入讲**！

```
"Scales in sixths"
        ↓
    Andrew 说要讲，但**没具体讲**
            ↓
    本笔记**补充**
```

### 基本定义（补充）

```
六度音阶：双音，每对相距 6 度
        ↓
    例（C 大调）：
    [C+A] - [D+B] - [E+C] - [F+D] - ...
        ↓
    每对都是"6 度"（实际是大六度或小六度）
```

### 与三度的对比

```
三度音阶：
- 跨度：3 度
- 手张开：略大
- 听感：饱满

六度音阶：
- 跨度：6 度
- 手张开：极大
- 听感：宽阔
        ↓
    六度对**手大**的人更友好
```

### 实战意义

```
Mozart / Schubert 钢琴曲：
- 大量六度并行
- 模仿"双声部"歌唱
        ↓
    练习六度 → 直接受益
```

## 5. 音阶的"方向感"和"力度"

[03:45] **乐感训练**！

### Andrew 的"音乐性"理念

[04:18] *"Every second you're at the piano you have to be practicing making music."*

```
经典错误：
- 练音阶时只想"音符正确"
- 完全无表情
- 像打字机
        ↓
    "在练习不流畅"
            ↓
    永远练不出"音乐"
```

### "方向"的概念

[04:00] **核心概念**！

```
Direction（方向）：
- 音阶**朝某个目标**走
- 通常：高点（climax）
- 然后**回家**（终点）
        ↓
    像"上山看日出，再下山"
```

### 力度处理

```
标准音阶力度：
1. 起点：mp（中弱）
2. 上行：渐强（crescendo）
3. 高点（顶 C）：f（强）
4. 下行：渐弱（decrescendo）
5. 终点：mp（中弱）
        ↓
    像"波浪"
            ↓
    比"全部相同"美 100 倍
```

### Andrew 的演示

[05:35] **对比**：

```
机械版：
[噔噔噔噔噔噔噔噔]
- 像电脑发声
- 无聊

音乐版：
[噔 噔 噔～ 噔～ 噔～！噔～ 噔～ 噔]
- 有起伏
- 有目标
- 听众喜欢
        ↓
    "Make music, not noise."
```

### 长 crescendo 的训练价值

[06:38] **关键技能**：

```
"Doing a long crescendo line and a long decrescendo line."
        ↓
    实战需要：
    - 整页缓慢变化
    - 1 小节 1 小节增强
    - 直到"高潮"那一拍
            ↓
    没练 → 永远做不到
            ↓
    练音阶时就练
```

## 6. 公式音阶（Formula Patterns）

[07:18] **本课最难的概念**！

### 名字解释

[07:23] *"Formula patterns sound so scientific, but it's really not."*

```
"Formula"听起来很学术
- 像"公式"
- 像"配方"
        ↓
    实际上：
    平行运动 + 反向运动 的"组合"
            ↓
    所以叫"公式"
```

### Andrew 的"舞蹈"教学法

[08:34] **创意**：

```
不在键盘上：
1. 双手在桌上
2. 都向上 1 八度
3. 反向（左手向左，右手向右）1 八度
4. 反向回来 1 八度
5. 都向上 1 八度
6. 都向下 1 八度
7. 反向 1 八度
8. 都向下 1 八度
        ↓
    "Middle - Out - Middle - Up - Middle - Down - Middle - Out - Middle - Down"
            ↓
    像跳舞
```

### 完整 8 步序列

[09:38] **公式音阶**：

```
标准 C 大调公式音阶：

第 1 步：双手平行向上 1 八度
        左手：C → C↑ ；右手：C → C↑

第 2 步：双手反向向外 1 八度
        左手：↓C ；右手：↑C

第 3 步：双手反向向内 1 八度
        左手：↑C ；右手：↓C

第 4 步：双手平行向上 1 八度
        左手：↑C ；右手：↑C

第 5 步：双手平行向下 1 八度
        左手：↓C ；右手：↓C

第 6 步：双手反向向外 1 八度
        左手：↓C ；右手：↑C

第 7 步：双手反向向内 1 八度
        左手：↑C ；右手：↓C

第 8 步：双手平行向下 1 八度
        左手：↓C ；右手：↓C
        ↓
    完整一遍 = 8 步
```

### 训练目标

```
公式音阶训练：
1. **手独立性**（极强）
2. **方向感**（每步换方向）
3. **速度**（节奏稳）
4. **平行 + 反向**协调
        ↓
    比普通音阶难 5 倍
            ↓
    专业钢琴家**必练**
```

### Andrew 的"分手关系"比喻

[11:21] **生动**：

```
两手 = 一对"朋友 / 情侣"
- 平行运动 = "我们一起走"（恋爱）
- 反向运动 = "我不喜欢你了"（分手）
- 平行回来 = "和好"
- 又反向 = "又吵"
        ↓
    8 步 = 一段"小连续剧"
            ↓
    "在一起 → 分手 → 和好 → 再分手"
```

## 7. 复节奏音阶（Polyrhythm Scale）

[13:08] **罕见但有趣**！

### 基本定义

```
复节奏音阶：
- 左手 2 连音 + 右手 3 连音（或反之）
- 一边弹音阶一边练复节奏
        ↓
    把第 26 课的复节奏 + 普通音阶 = 复节奏音阶
```

### Andrew 的口诀

[13:34] **来自第 26 课**：

```
"Not difficult"（3:2 复节奏口诀）
        ↓
    用在音阶上：
    左手：每拍 2 个音
    右手：每拍 3 个音
            ↓
    边弹边念"not difficult"
```

### 视觉对照

```
普通音阶：双手同步
- 左：C-D-E-F-G-...
- 右：C-D-E-F-G-...

复节奏音阶：
- 左：C-D-E-F-G-A-B-C（2 连音）
- 右：C-D-E-F-G-A-B-C-D-E-F-G（3 连音）
        ↓
    右手"超前"于左手
            ↓
    每拍**双手不齐**
    但每"小节"双手回到同一个 C
```

### 训练目标

```
复节奏音阶训练：
1. 复节奏稳定
2. 手独立性
3. 高难协调
        ↓
    比公式音阶**还难**
            ↓
    高级 / 比赛 / 大学考试
```

### Andrew 的诚实

[14:43] **承认**：

```
"That actually wasn't the best for ratio."
        ↓
    Andrew 自己演示**不完美**
    复节奏音阶**真的难**
            ↓
    给学生看到"真实状态"
```

## 8. 重复音音阶（Repeated Note Scale）

[15:32] **解决"弱调"利器**！

### 基本定义

```
重复音音阶：
- 每个音弹**3 次**（3-2-1 指法）
- 然后移到下个音
        ↓
    例（C 大调）：
    C-C-C - D-D-D - E-E-E - F-F-F - ...
            ↓
    每个音用 3-2-1 三个手指轮换
```

### 指法

```
重复音音阶的指法：
3 - 2 - 1 - 3 - 2 - 1 - 3 - 2 - 1 - ...
        ↓
    用 3 → 2 → 1 三指**轮换**
    不是同一指弹 3 次！
            ↓
    像"接力赛"
```

### 训练目标

[15:49] **核心目的**！

```
"This is great for repeated notes and improving weaker scales."
        ↓
    重点 1：训练**重复音**技术
    重点 2：**强化薄弱调**
        ↓
    哪个调你弹得不流畅？
    用重复音音阶强化它
```

### 实战应用

[17:38] **真实曲目**！

```
Beethoven《Für Elise》：
- 多处重复音
- 需要"3-2-1"指法

Liszt：
- 大量重复音段落
- 极快速度

Maurice Ravel《Alborada del gracioso》：
- 著名重复音段
- 极难
        ↓
    重复音 = 浪漫派常用
```

### 演奏要点

```
1. 开始**慢速**（60 BPM）
2. 每个手指**等响**
3. 不要**用力过度**
4. 渐渐**加速**
        ↓
    100-120 BPM 是目标
```

## 9. RCM 等级要求（加拿大）

[12:07] **重要标准**！

### RCM 是什么？

```
RCM = Royal Conservatory of Music
（加拿大皇家音乐学院）
        ↓
    加拿大钢琴等级考试机构
    全球认可
            ↓
    类似中国的"中央音乐学院考级"
    类似英国的"ABRSM"
```

### 各等级的音阶要求

```
RCM 等级 1-3：
- 单声部音阶
- 简单调（C / G / F）

RCM 等级 4-6：
- 三度音阶（部分调）
- 公式音阶基础

RCM 等级 7-9：
- 完整三度音阶（所有大小调）
- 公式音阶（所有大小调）
- 反向音阶

RCM 等级 10：
- **每个**音阶**每个**调
- 三度 + 六度 + 八度 + 公式 + 反向
- 加上**减七**和**属七**琶音
        ↓
    专业级别要求
```

### Andrew 的"等级 10"提醒

[12:14] *"In Canada we have the RCM, and in grade 10 you're supposed to do every single scale contrary motions."*

```
等级 10 的"完整要求"：
- 12 个大调 × 每种音阶
- 12 个小调 × 每种音阶
- 三种小调（自然 / 和声 / 旋律）
- 总共 36 个调 × 4-5 种音阶
- = **144+ 个音阶**
            ↓
    标准 1-2 年准备
```

### 国际等级对照（补充）

```
RCM (加拿大) ↔ ABRSM (英国) ↔ Trinity (英国)：
RCM 1 ≈ ABRSM 1
RCM 5 ≈ ABRSM 5
RCM 8 ≈ ABRSM 8 (准入大学)
RCM 10 ≈ ABRSM Diploma
            ↓
    国际通用
```

## 14. 双校准：与通用钢琴教学对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **三度音阶定义** —— 标准。
2. **每天 10-20 分钟** —— 合理建议。
3. **音乐性 > 速度** —— 标准教学。
4. **公式音阶定义** —— 标准（叫 "Formula Pattern"）。
5. **复节奏音阶** —— 准确（与第 26 课呼应）。
6. **重复音 3-2-1 指法** —— 标准。
7. **RCM 等级 10 要求** —— 准确。
8. **找好老师重要** —— 标准建议。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[01:35] "Start with E down or sorry that's a six"** —— Andrew 自我修正。
   → **校准**：不影响教学。

2. **[02:00] "These are an octave apart"** —— 视频简化。
   → **校准**：实际 C 到 C 是 8 度。

3. **[06:38] "Long crescendo line"** —— Andrew 简化。
   → **校准**：完整术语为 "molto crescendo poco a poco"（一点一点逐渐增强）。

4. **[09:38] 公式音阶的 8 步**：Andrew 念得稍乱。
   → **校准**：本笔记**整理**了清晰的 8 步序列。

5. **[14:43] 复节奏音阶演示不完美**：
   → Andrew 自己说"sorry"。
   → 给学生看到"真实"。

6. **[18:26] "2.2 out of 10"**：
   → Andrew 演示重复音音阶时极度自嘲。
   → 学生可能怀疑老师水平。

7. **没讲六度音阶的具体方法**：
   → Andrew 提了，但没演示
   → 本笔记补充

8. **没讲八度音阶**：
   → 等级 8+ 必备
   → 视频跳过

9. **没讲减七和弦琶音**：
   → 等级 9-10 要求
   → 视频跳过

### ❌ 视频中的小错误

1. **[02:54] "Try to make both your hands go in a straight line — level"** —— 表达不严谨。
   → **校准**：实际是"力度均匀"，不是"物理水平线"。

2. **[12:14] "Grade 10 you're supposed to do every single scale contrary motions"** —— 不全。
   → **校准**：等级 10 还要求**反向**音阶（contrary motion scale）的具体形式。

3. **[16:18] "My fingernail got chopped off"** —— 个人细节。
   → 与教学无关，但可以理解为"老师也是人"。

4. **没讲"小调音阶的特殊"**：
   → 三种小调（自然 / 和声 / 旋律）在三度 / 六度音阶中**不同**
   → 视频仅讲大调

5. **没讲"减音阶"**（Diminished Scale）和"全音阶"（Whole Tone Scale）：
   → 现代音乐和爵士常用
   → 视频未提

6. **没讲"五声音阶"**（Pentatonic）：
   → 流行音乐和东方音乐基础
   → 视频未提

### 💡 视频遗漏的重要内容

1. **小调三种音阶在三度上的差异**：
   - 自然小调三度音阶
   - 和声小调三度音阶
   - 旋律小调三度音阶
   - 视频跳过

2. **八度音阶**：
   - RCM 等级 7-8 要求
   - 视频跳过

3. **减七和属七琶音**：
   - RCM 等级 9-10 要求
   - 视频跳过

4. **现代音乐音阶**：
   - 爵士（Mixolydian / Dorian）
   - 蓝调
   - 五声
   - 全音
   - 减音
   - 视频未提

5. **东方音阶**：
   - 中国五声 / 印度 raga
   - 视频零提及

6. **Cortot《Rational Principles》**：
   - 包含**所有**音阶变体
   - 比哈农更现代
   - 视频未推荐

7. **录音对比**：
   - Glenn Gould 的音阶处理（极平稳）
   - Vladimir Horowitz 的音阶处理（极有起伏）
   - 听不同风格
   - 视频未推荐

8. **现代练习 App**：
   - **Modacity** 智能练习
   - **Tonic** 节拍 + 音阶
   - 视频是 2008 年作品

9. **健身手指的"3 大原则"**：
   - 平衡 / 放松 / 控制
   - 与第 34 课哈农呼应
   - 视频未明确总结

10. **公式音阶的"标准谱面"**：
    - 视频用"舞蹈"代替谱面
    - 学生看不到具体写法
    - 本笔记**补充**了 8 步清单
