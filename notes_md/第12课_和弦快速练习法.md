## 2. 和弦的构造原理：用音程"叠"起来

[01:21] *"All chords are built on minor and major thirds."*

### 核心原则

> 🎯 **所有和弦 = 一连串大三度（M3）和小三度（m3）的叠加**

### 大三度（M3） vs 小三度（m3）回顾

| 音程 | 半音数 | 例子 |
|---|---|---|
| **大三度（M3 = +）** | 4 个半音 | C → E |
| **小三度（m3 = −）** | 3 个半音 | E → G |

### Andrew 的"电池"比喻

[02:08] *"Think of plus as the major third and minus as the minor third — like a battery."*

```
M3 = + (正极，明亮)
m3 = − (负极，忧郁)

不同的 +/- 排列组合 = 不同的和弦音色
```

---

## 3. 大三和弦 vs 小三和弦（复习）

### 大三和弦 (Major Triad) = M3 + m3

```
C 大三和弦：
  C ──[M3 = 4 半音]── E ──[m3 = 3 半音]── G
  ↑                                       ↑
  根音                                    五音
```

例子：
- **C 大**：C – E – G
- **G 大**：G – B – D
- **F 大**：F – A – C

### 小三和弦 (Minor Triad) = m3 + M3（顺序反过来！）

```
A 小三和弦：
  A ──[m3 = 3 半音]── C ──[M3 = 4 半音]── E
```

例子：
- **A 小**：A – C – E
- **D 小**：D – F – A
- **E 小**：E – G – B

### 听感对比

| 类型 | 公式 | 听感 | 助记 |
|---|---|---|---|
| **大三和弦** | M3 + m3 | 明亮、欢快 | "正能量" |
| **小三和弦** | m3 + M3 | 忧郁、深沉 | "负能量" |

## 4. 四音和弦（Four-Note Chord）

[02:24] **三和弦的"加八度"版本**：在三和弦顶上加一个**和根音相同**的高八度音。

### 构造

```
三和弦： C – E – G          （3 个音）
四音：   C – E – G – C(↑8)   （4 个音）
                       ↑
                   和根音相同，高一个八度
```

### 推荐指法

[02:34] **右手指法**：1 – 2 – 3 – **5**

```
拇指 (1)：根音 C
食指 (2)：三音 E
中指 (3)：五音 G
小指 (5)：八度 C
```

### 手够不到怎么办？

[02:35] *"If your hand can't reach it, twist your hand a little bit to the side."*

→ **手稍向外旋转**（不是上下），让小指能伸到八度上。

> 💡 **别强求**：手小的人（如 5'4" 以下女性、儿童）八度可能弹不到。**先用三和弦**练，等手长大或拉伸后再加八度。

## 5. 属七和弦（Dominant 7th Chord）

[04:33] *"The formula to build the dominant 7th chord is major-minor-minor."*

### 构造公式：M3 + m3 + m3

```
C 属七和弦 (C7)：
  C ──[M3]── E ──[m3]── G ──[m3]── B♭
  +          −           −
  
  4 个音：C, E, G, B♭
```

### 关键差异：B♭ vs B

| 大三和弦 (C) | 大七和弦 (Cmaj7) | 属七和弦 (C7) |
|---|---|---|
| C – E – G | C – E – G – **B** | C – E – G – **B♭** |
| 完整 | 高雅、爵士味 | **想回家**的张力 |

→ **属七的灵魂在于这个"降七"**！它制造"不稳定"，逼着耳朵期待解决到主和弦。

### 属七的功能：解决到主和弦

[05:35] *"A dominant 7th chord leads you back to the tonic."*

```
G7 → C    （G7 是 C 大调的属七，"回家"到 C 大）
        ↓ 经典进行
        听感：紧张 → 释放
```

实例：
- **F 大调中**：C7 → F（C7 是 F 的属七）
- **G 大调中**：D7 → G
- **A 小调中**：E7 → Am（小调中也用！）

### 命名"属"的来源

属（dominant）= 调中的**第 5 音**。从第 5 音建立的七和弦 = **属七和弦 (V7)**。

例：
- C 大调的属音 = G → C 大调的属七 = **G7**（G – B – D – F）
- F 大调的属音 = C → F 大调的属七 = **C7**

> ⚠️ **注意**：本课 Andrew 全部用 **C 作为根音**演示，但**实际的 C7 是 F 大调的属七**！要记得"属七"的命名是基于**它在哪个调里担任 V 级**，而不是它从哪个音开始。

## 6. 减七和弦（Diminished 7th Chord）

[06:50] *"Minor-minor-minor — that's a diminished 7th chord."*

### 构造公式：m3 + m3 + m3

```
C 减七和弦 (C°7 / Cdim7)：
  C ──[m3]── E♭ ──[m3]── G♭ ──[m3]── B♭♭(=A)
  −           −            −
  
  4 个音：C, E♭, G♭, A (技术上是 B♭♭)
```

### 听感

[07:33] *"It kind of has a spooky sound to it."*

→ 听起来**恐怖、悬疑、不安**。莫扎特、贝多芬、肖邦都常用，但**特别是恐怖电影配乐**最爱。

### Andrew 的拼写小错误

视频中 Andrew 用 G♭ 和 A 演示。严格写法应为 **B♭♭**（双降 B），但 **A 和 B♭♭ 是同一个键**（异名同音）。
→ 演奏上**完全相同**，记谱时为遵循"减七全是小三度"的规则，正式应写双降。本笔记保留 A 这种"实用拼写"。

### 减七的"循环"特性（极重要！）

[08:14] *"Actually there's only three diminished 7ths in existence!"*

#### 为什么只有 3 种？

减七全是小三度（3 半音），4 个音叠加共跨**12 个半音 = 1 个八度**。
所以**任意一个减七和弦的转位 = 同一个减七**！

```
C – E♭ – G♭ – A
     ↓ 转位（A 提到顶部 + 八度）
E♭ – G♭ – A – C
     ↓ 再转位
G♭ – A – C – E♭
     ↓ 再转位
A – C – E♭ – G♭
     ↓ 转位回到原位
C – E♭ – G♭ – A
```

### 全部 3 种减七和弦

```
1. C – E♭ – G♭ – A   （= C°7 = E♭°7 = G♭°7 = A°7）
2. C♯ – E – G – B♭   （= C♯°7 = E°7 = G°7 = B♭°7）
3. D – F – A♭ – B    （= D°7 = F°7 = A♭°7 = B°7）
```

→ **整个键盘上只有 3 个"音组合"算减七**，覆盖所有 12 个调！

### 实战应用：贝多芬《月光奏鸣曲》第二页

[07:52] *"Did you know all those notes are comprised of different diminished 7th chords?"*

→ 月光奏鸣曲第一乐章中段那段经典的"上下波动"，**全部由减七和弦构成**！这就是它"诡异、压抑"的来源。

## 7. 4 种和弦构造公式总结

> 🎯 **必背**！这是和弦学习的"四件套"。

| 和弦类型 | 公式 | C 根音例子 | 听感 |
|---|---|---|---|
| **大三** (Maj) | M3 + m3 | C – E – G | 明亮 |
| **小三** (min) | m3 + M3 | C – E♭ – G | 忧郁 |
| **属七** (7) | M3 + m3 + m3 | C – E – G – B♭ | 想回家 |
| **减七** (°7) | m3 + m3 + m3 | C – E♭ – G♭ – A | 恐怖 |

### 记忆口诀

```
大三：+  −           （明 + 暗）
小三：−  +           （暗 + 明）
属七：+  −  −         （明 + 暗 + 暗）
减七：−  −  −         （暗 + 暗 + 暗）
```

> 💡 **观察**：每多一个 m3，就更"暗" / 更"紧张"。  
> **大三 → 小三 → 属七 → 减七 = 从最亮到最暗**

## 8. 核心练习法：半音上行 + 同型移位

[10:18] **本课最重要的练习方法！**

### 核心思想

**所有和弦的"形状"是固定的**。把这个形状**整体平移半音**，就得到下一个调的同型和弦。

### 实战流程（以大三和弦为例）

```
第 1 个：C 大三  ：C  – E  – G  – C
                   ↓ 整体上移半音
第 2 个：C♯ 大三：C♯ – F  – G♯ – C♯
                   ↓ 整体上移半音
第 3 个：D 大三  ：D  – F♯ – A  – D
                   ↓ ...
                   
共 12 个调 × 1 八度 + 回到起点 = 13 个和弦
```

### 完整练习（12 个半音 = 12 个大三）

```
C → C♯ → D → D♯ → E → F → F♯ → G → G♯ → A → A♯ → B → C(高 8 度)

然后下行：
C(高) → B → A♯ → A → ... → C
```

### 这个练习的"魔法"

[15:01] *"You can learn every single note, every key in one exercise."*

| 优点 | 解释 |
|---|---|
| **不需要记调号** | 每个调的"形状"一样，靠肌肉记忆 |
| **手指自动学会所有调** | 从 C 大调到 F♯ 大调，无差别训练 |
| **和弦反应速度** | 喊出"E♭ 大"，手立刻找到位置 |
| **建立"和弦地图"** | 大脑里浮现所有 12 个调的位置 |

### 练习时间分配

[20:00] *"Don't go hardcore. Go up and down maybe 5-10 times a day."*

```
建议：
- 早：上下大三和弦 5 次（约 5 分钟）
- 中：上下小三和弦 5 次
- 晚：上下属七和弦 + 减七和弦各 5 次
                ↓
            每天约 20 分钟
            坚持 4 周 → 所有调的所有和弦都熟
```

### Andrew 老师的"我会今天学这个和弦"心象法

[18:08] Andrew 提到他的老师 Sherry-Lee Elder（"thank you, Lori"）教过的方法：

```
弹每个和弦时，心中默念：
"I will learn this chord today. I will learn this chord today."
```

→ **意识 + 动作同时投入** = 学得更快、更深。

## 9. 4 种触键变化练习

[16:30] 同一个练习，用 4 种触键各练一遍 = 4 倍价值。

### 1. 实心和弦（Block Chord）

```
"咣！" 一下整个和弦同时按下，再放开，下一个
   ↓
弹奏：[和弦] - 停 - [和弦] - 停 - ...
```

适合：练**手型固定**和**音色融合**。

### 2. 断奏（Staccato）

```
"哒哒哒哒"，每个和弦短促有力
弹奏：⚫⚫·  ⚫⚫·  ⚫⚫·  ...
```

适合：练**手腕弹性**和**抬键速度**。

### 3. 节奏型（Rhythmic Patterns）

```
慢-快-快  慢-快-快
[和弦] [和弦][和弦]  [和弦] [和弦][和弦]
```

或：

```
强-弱-中-弱（4/4 律动）
[强][弱][中][弱]  [强][弱][中][弱]
```

适合：练**力度控制**和**节奏感**。

### 4. 琶音（Rolled Chord）

[17:40] *"You roll a chord — the wrist sets up the motion."*

```
[根]→[三]→[五]→[八]
四个音**先后**弹出（极快），但**手指都按下不放开**

听感：哆-哆-哆-哆 → 形成"竖琴式"音色
```

#### 怎么"滚"？

```
1. 手腕**前倾**
2. 用**手腕的弧线运动**带动手指依次落键
3. 注意：每个手指落键后**不松开**
4. 最终所有 4 音同时持续响
```

适合：练**手腕灵活性**和**和弦的"涟漪感"**。

## 10. 触键技巧：圆手 + 顶音突出

### 圆手（Round Hand）

[05:18] *"My hand is round. I'm just letting my fingers go right down to the bottom of the note."*

```
错误：手指伸直，像拍打 ❌
                ↓ 力度集中在手腕
                ↓ 容易弹错音 + 手指疲劳
                
正确：手指弯曲如握鸡蛋 ✅
                ↓ 力度自然集中到指尖
                ↓ 准确度高 + 手指放松
```

### 整体落键（Solid Block）

[02:48] *"What you want to aim with all chords is that they're going down solid."*

```
错误：四音错落响起 ❌（哆-哆-哆-哆）
正确：四音**同时**响起 ✅（哐！）
```

### 突出顶音（Voice the Top Note）

[16:09] *"With all chords, you actually want to voice the top note."*

```
四音和弦：C – E – G – C(↑)
                       ↑
                   小指弹的这个 C 略响一些！
                   
听感：底部三音是"和声底色"，顶音是"主旋律"
```

#### 为什么？

钢琴和弦中，**最高音**是听众**最容易听到**的。把它弹得稍响 = **旋律线条清晰** = 演奏更有"音乐性"。

### 不要"砸"（No Clunking）

[16:02] *"Try to push forward, don't just clunk down."*

```
错误：从天而降砸下去 ❌（咣！冷硬）
正确：手指水平**推进**琴键 ✅（柔软渗透）
```

## 13. 双校准：与通用乐理对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **大三和弦 = M3 + m3 / 小三和弦 = m3 + M3** —— 标准定义。
2. **属七和弦 = M3 + m3 + m3** —— 正确。
3. **减七和弦 = m3 + m3 + m3** —— 正确。
4. **属七的"想回家"功能 (V7 → I)** —— 调性音乐的核心进行，标准。
5. **减七和弦只有 3 种（按音组合）** —— 正确（这是减七和弦的"对称性"特征）。
6. **贝多芬月光奏鸣曲中段使用减七和弦** —— 历史属实。
7. **半音上行练习法** —— Hanon 等钢琴教材的标准训练方法。
8. **顶音突出原则 (voicing the top)** —— 现代钢琴技巧的核心教导。
9. **圆手 + 整体落键** —— 标准技术。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[06:50] "G♭ and A" 在减七中** —— 严格说应该是 **G♭ 和 B♭♭（双降 B）**，因为减七要求**全是小三度**。  
   → **澄清**：演奏上 A = B♭♭（异名同音），但**正式记谱**用 B♭♭ 才符合规则。本笔记已注明。

2. **[03:00] "Don't ever do that"**（演示错落弹奏时）—— Andrew 表达过激。  
   → **校准**：错落弹奏（琶音）**也是合法触键**，是"另一种音色"，不是"错"。第 9 节已重新整理为 4 种触键之一。

3. **[09:13] "Diminished means small. Augmented is bigger"** —— 简化表达。  
   → **更准确**：

### 💡 视频遗漏的重要内容

1. **大七和弦 (Maj7)**：公式 = M3 + m3 + M3。例：Cmaj7 = C – E – G – **B**（不是 B♭！）。  
   → 大七是爵士、流行音乐**最常用**的和弦之一，视频**没讲**。

2. **小七和弦 (m7)**：公式 = m3 + M3 + m3。例：Cm7 = C – E♭ – G – B♭。  
   → 小调中的常见和弦，视频**没讲**。

3. **半减七和弦 (m7♭5 / ø7)**：公式 = m3 + m3 + M3。例：Cm7♭5 = C – E♭ – G♭ – B♭。  
   → 介于减七和小七之间，视频**没讲**。

4. **七和弦总表（爵士基础）**：

| 类型 | 公式 | C 根音 |
|---|---|---|
| 大七 (Maj7) | M3 + m3 + M3 | C – E – G – B |
| 小七 (m7) | m3 + M3 + m3 | C – E♭ – G – B♭ |
| 属七 (7) ✓ | M3 + m3 + m3 | C – E – G – B♭ |
| 半减七 (m7♭5) | m3 + m3 + M3 | C – E♭ – G♭ – B♭ |
| 减七 (°7) ✓ | m3 + m3 + m3 | C – E♭ – G♭ – A |

→ Andrew 只讲了打勾的两种。**完整 5 种七和弦**才能覆盖大部分音乐。

5. **增三和弦 (Augmented Triad)**：公式 = M3 + M3。例：C+ = C – E – G♯。  
   → 罕见但重要（如肖邦、印象派常用），视频**完全没提**。

6. **和弦转位**：第 3 课讲过转位概念，但**七和弦的转位**（共 4 种：原位 / 第一转位 / 第二转位 / 第三转位）视频没展开。

7. **大调音阶上每个音的七和弦**（"七和弦在调内"）：C 大调内的 7 个七和弦：
   - I: Cmaj7
   - ii: Dm7
   - iii: Em7
   - IV: Fmaj7
   - V: G7（视频讲的属七！）
   - vi: Am7
   - vii°: Bm7♭5

   → 这是**学和声的关键步骤**，视频未涉及。
