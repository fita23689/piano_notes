## 1. 本课概览

| 终止式 | 和弦走向 | 听感 | 标点 |
|---|---|---|---|
| **完全终止** | V → I | 完美结束 | 句号 |
| **变格终止** | IV → I | 教堂结束 | 句号 / 逗号 |
| **半终止** | I → V | 暂时停顿 | 问号 |
| **阻碍终止** | V → vi | 意外转折 | 感叹号 + 问号 |

> 💡 终止式是音乐的"标点符号"——决定**乐句怎么结束**。

## 2. 什么是终止式（Cadence）

[00:11] *"Think of music as a sentence. A cadence is a punctuation."*

### 核心定义

```
音乐 = 句子
终止式 = 标点符号
        ↓
    乐句的"结尾标志"
    告诉听众："这一段结束了"
```

### 文字 vs 音乐

```
文字标点：
- 句号 (.)：陈述结束
- 问号 (?)：疑问
- 感叹号 (!)：惊讶
- 逗号 (,)：暂停
        ↓
音乐终止式：
- 完全终止：陈述结束（句号）
- 半终止：疑问（问号）
- 阻碍终止：惊讶（感叹号）
- 变格终止：温和结束（逗号 / 句号）
```

### 终止式的功能

```
1. 结构标记：告诉听众"乐段在哪结束"
2. 情感传达：根据类型传达不同情绪
3. 期待营造：让听众"预测"接下来发生什么
4. 调性确立：通过 V → I 确认主调
            ↓
    没终止式 → 音乐"无结构"
    像没标点的文章
```

## 3. 完全终止（Perfect Cadence）

[01:09] **最重要的终止式**！

### 和弦走向

```
V → I（5 → 1）
或
V7 → I（5-7 → 1）
        ↓
    例（C 大调）：
    G - C
    或
    G7 - C
```

### 实战例子

```
C 大调中：
- I: C - E - G
- V: G - B - D
- V7: G - B - D - F

V → I：
[G B D] → [C E G]
        ↓
    导音 B → 主音 C
    属音 D → 主音 C / 中音 E
    根音 G → 主音 C / 中音 E
```

### 听感

[02:00] *"It sounds finished — like grabbing the donut you wanted."*

```
听感：
✅ "完成"
✅ "落下"
✅ "圆满"
✅ "终于到了"
            ↓
    像数学题"=" 之后的答案
```

### 为什么 V → I 给"结束感"？

```
理论原因：
1. 导音解决：B → C 是最强解决
2. 三全音解决：F → E + B → C（V7 中）
3. 根音五度跳进：G → C 是最自然的进行
            ↓
    所有"张力"都解决了
    听众"放心"
```

### 在乐曲中的位置

```
✅ 全曲结尾：必须用
✅ 乐章结尾：通常用
✅ 主题结尾：常用
✅ 短乐句：偶尔
            ↓
    "完美句号"
```

### Andrew 的两种演示

[01:50] 视频中：

```
版本 A（V7 → I）：
G B D F → C E G C
        ↓
    更"丰满"

版本 B（V → I）：
G B D → C E G
        ↓
    更"简洁"
```

## 4. 变格终止（Plagal Cadence）

[03:24] **"教堂终止"**！

### 和弦走向

```
IV → I（4 → 1）
        ↓
    例（C 大调）：
    F - C
```

### 实战例子

```
C 大调中：
- I: C - E - G
- IV: F - A - C

IV → I：
[F A C] → [C E G]
        ↓
    根音 F → C（下行 5 度）
    顶音 C 共同
    中音 A → G（下行 2 度）
```

### 听感

[03:30] *"They have a more lovely sound — they remind me of a church."*

```
听感：
✅ "温和"
✅ "教堂感"
✅ "柔和落下"
            ↓
    像基督教礼拜的"Amen"结尾
```

### 著名例子：Amen 终止

```
基督教礼拜结束：
"Amen"
↓
和声：
[F A C] → [C E G]
A     - men
        ↓
    几乎所有教堂赞美诗都用变格终止
    所以"Plagal Cadence"也叫"Amen Cadence"
```

### Andrew 的"次要"评价

[04:30] *"A plagal cadence isn't quite as accepted as the true way to end. The perfect cadence is the favorite child."*

```
Andrew 的看法：
- 完全终止：长子（最重要）
- 变格终止：次子（替代 / 补充）
            ↓
    在大型作品中：
    - 主要结束 → 完全终止
    - 装饰结束 → 变格终止
```

> ⚠️ **校准**：Andrew 的"次要"评价**部分准确**，但变格终止在浪漫派以后**地位平等**。

## 5. 半终止（Imperfect / Half Cadence）

[05:01] **"问号终止"**！

### 和弦走向

```
任何和弦 → V（结束在 V 上）
        ↓
    最常见：I → V
    或：IV → V、ii → V 等
        ↓
    例（C 大调）：
    C - G（结束在 G）
```

### 听感

[05:09] *"It propels the piece forward — like the opposite of perfect."*

```
听感：
✅ "未完成"
✅ "悬而未决"
✅ "向前推动"
            ↓
    像问句的"?"
    听众期待"答案"
```

### 在乐曲中的位置

```
✅ 乐句中段
✅ 主题展开
✅ 引出下一段
❌ 全曲结尾（不行！）
            ↓
    "半句话"，需要"另一半"
```

### Andrew 的两种演示

[05:39] 视频中：

```
版本 A（I → V）：
[C E G] → [G B D]
        ↓
    最简单

版本 B（IV → V）：
[F A C] → [G B D]
        ↓
    更有"推进感"
```

### 为什么叫"Imperfect"？

```
"Imperfect" = "不完美"
        ↓
    "不完美"的对象：完全终止
    没"完美"地结束
        ↓
    与中文"半终止"对应
```

> ⚠️ **校准**：Andrew 用 "Imperfect Cadence" 一词，但**现代英语教学**更常用 "Half Cadence"。两者意思相同。

## 6. 阻碍终止（Deceptive Cadence）

[06:18] **"惊喜终止"**！

### 和弦走向

```
V → vi（5 → 6m）
不是 → I！
        ↓
    例（C 大调）：
    G - Am（不是 C！）
```

### 听感

[06:38] *"You feel like it's about to finish — and then, oh, maybe not."*

```
听感：
✅ "本以为结束"
✅ "意外转折"
✅ "戏剧性"
            ↓
    像"看到甜甜圈伸手 → 妈妈打手"
```

### Andrew 的"甜甜圈被打手"

[06:48] **生动比喻**：

```
完全终止（V → I）：
- 看到甜甜圈
- 伸手拿到
- 吃了 → "Yay!"

阻碍终止（V → vi）：
- 看到甜甜圈
- 伸手拿
- 妈妈打手 → "No!"
- 听众："啊？"
            ↓
    "戏剧性反转"
```

### 实战例子

```
C 大调中：
- V: G - B - D
- vi: A - C - E

V → vi：
[G B D] → [A C E]
        ↓
    根音 G → A（上行 2 度）
    第三 B → C（上行半音）
    第五 D → E（上行 2 度）
            ↓
    所有声部**上行**
    给听众"被推开"感
```

### 在乐曲中的位置

```
✅ 高潮段：增加戏剧
✅ 假结尾：欺骗听众
✅ 重复片段：第一次假结尾，第二次真结尾
            ↓
    "戏剧化"作曲家最爱
    Beethoven、Brahms 大量使用
```

### 历史上的著名例子

```
Beethoven《第 9 交响曲》第 1 乐章：
- 多次"伪结束"
- 最后才真结束
            ↓
    "等待已久的解脱"
```

## 7. 开放 vs 封闭终止

[08:46] **进阶分类**！

### 封闭终止（Closed Cadence）

```
定义：完全终止 + 主调 + C 在最高音
        ↓
    例（C 大调）：
    [G B D] → [C E G C] (高音 C)
            ↓
    "完全的"完全终止
    全曲终结的"必杀技"
```

### 开放终止（Open Cadence）

```
定义：以下任何一项不满足封闭条件
- 不在主调（如转到了别的调）
- 高音不是主音（如高音是 E 或 G）
- 不是完全终止
        ↓
    例（C 大调）：
    [G B D] → [C E G] (高音 G)
            ↓
    虽然是 V → I，但**开放**
    暗示"还会继续"
```

### Andrew 的"盒子"比喻

[10:34] *"With C on the top and bottom, it's like closing the box on both sides — saying 'you're not getting out.'"*

```
封闭终止：
- 最高音 C + 最低音 C
- 像"封闭的盒子"
- 听感"完全终止"

开放终止：
- 最高音 E（或 G）
- 像"未关的盒子"
- 听感"还能溜出去"
            ↓
    高音的位置决定"封闭度"
```

### 实战意义

```
作曲应用：
- 全曲结尾：用封闭
- 乐章结尾：用封闭
- 乐段结尾：用开放（让人继续听）
- 主题展开：用开放
            ↓
    控制听众的"等待感"
```

## 8. 声部进行（Voice Leading）

[12:12] **本课的"技术核心"**！

### 什么是声部进行？

```
声部进行 = 各声部如何"流畅地"从一个和弦到下一个
        ↓
    和弦中每个音 → 下个和弦中**相邻**音
    最少跳跃，最大流畅
```

### V → I 的"标准"声部进行

[12:14] **Andrew 演示**：

```
V (G B D) → I (C E G)：
- B → C（上行半音）✅
- D → C（下行 2 度）✅
- G → G（共同音保留）✅
            ↓
    最少跳跃，最流畅
```

### 4 个声部的对位

```
SATB（女高音 / 女低音 / 男高音 / 男低音）：

V → I 的标准对位：
S：D → C（下行）
A：G → G（保留）
T：B → C（上行）
B：G → C（下行 5 度，跳进）
            ↓
    Bass 可以跳进
    其他声部 → 流畅进行
```

### 流畅性的"金科玉律"

```
✅ 共同音 → 保留
✅ 相邻音 → 级进（半音或全音）
✅ 远音 → 反方向运动
❌ 平行 5 度 / 8 度 → 禁止（详见第 30 课）
            ↓
    声部进行是和声的"基本功"
```

### 实战练习

```
1. 写出 V → I（4 个声部）
2. 检查每个声部如何"走"
3. 修正不流畅的地方
4. 听感测试
            ↓
    1 周练习就有明显进步
```

## 9. Andrew 的"标点符号"比喻

[10:50] **整集最直观的比喻**！

### 完整对应表

| 终止式 | 标点 | 听感 | 用途 |
|---|---|---|---|
| **完全终止** | 句号 (.) | 完成 | 结束乐句 |
| **变格终止** | 句号 (.) 或 逗号 (,) | 温和 | 教堂式结束 |
| **半终止** | 问号 (?) | 疑问 | 推进 |
| **阻碍终止** | 感叹号 (!) | 惊讶 | 戏剧 |

### 实战应用

```
"今天天气真好。"
→ 完全终止

"今天天气真好，"
→ 变格终止 / 半终止

"今天天气真好？"
→ 半终止

"今天天气真好！"
→ 阻碍终止
            ↓
    根据"语气"选终止式
```

### 阅读音乐的"语法分析"

```
听陌生曲：
1. 找终止式
2. 判断类型
3. 知道乐句结构
            ↓
    像分析中文段落的"语法"
```

## 13. 双校准：与通用和声理论对照

> 🔬 **目标**：核对教学准确性，标注准确点 / 表达瑕疵 / 需要补充。

### ✅ 完全正确

1. **终止式 = 音乐标点** —— 经典比喻。
2. **完全终止 = V → I** —— 标准。
3. **变格终止 = IV → I** —— 标准。
4. **半终止 = 结束在 V 上** —— 标准。
5. **阻碍终止 = V → vi** —— 标准。
6. **导音 B → C 解决** —— 准确。
7. **封闭终止 = C 顶底** —— 准确。
8. **声部进行核心原则** —— 标准。

### ⚠️ 表达瑕疵 / 易混淆点

1. **[01:11] "Imperfect Cadence"** —— Andrew 用此名。
   → **校准**：现代英美教学更常用 "Half Cadence"。两者**完全等同**，都指"结束在 V 上"。

2. **[04:54] "Plagal is alternative"** —— **过度贬低**变格终止。
   → **校准**：
   - 巴洛克 / 古典：完全终止主导（Andrew 对）
   - 浪漫 / 现代：变格终止地位平等
   - 民间音乐：变格终止常用
   - 不是"次要"，是"风格选择"

3. **[06:30] "Deceptive Cadence"** —— Andrew 这个术语严格说**不全面**。
   → **校准**：
   - **Deceptive Cadence** (V → vi) ✅
   - **Interrupted Cadence** (V → 任何不是 I 的) — 更宽泛术语
   - 两者**部分重叠**

4. **[09:01] "Closed cadence has C on top and bottom"** —— 简化。
   → **校准**：
   - 严格说，"完美完全终止" (Perfect Authentic Cadence, PAC) 要求：
     - V → I（不是 V7 → I 一定要 PAC）
     - I 的根音在最高 + 最低声部
     - 用根位 V 和 I（不是转位）
   - 不只是"C 在顶底"

5. **[02:11] "5 chord to 1 chord"** —— Andrew 用"5"而不是 "V"。
   → **校准**：现代标准用罗马数字 V / IV / I。"5"是简化记法。

6. **没讲"完美完全终止" (PAC) vs "不完美完全终止" (IAC)**：
   → 两者都是"V → I"，但严格性不同
   → PAC：根位 + 主音在顶
   → IAC：转位或主音不在顶
   → 视频简化为"开放 / 封闭"。

### ❌ 视频中的小错误

### 💡 视频遗漏的重要内容

1. **完美完全终止 (PAC) vs 不完美完全终止 (IAC)**：
   - 现代和声教学的关键区分
   - 视频简化为"开放 / 封闭"
   - 不够精确

2. **Phrygian Cadence**（弗里几亚终止）：
   - iv6 → V
   - 巴洛克和古典常用
   - 听起来"古老 / 庄严"
   - 视频未提

3. **倍下属和弦终止**（Picardy Third）：
   - 小调中用大主和弦结束
   - 巴洛克常用
   - 听感"突然光明"
   - 视频未提

4. **教会调式终止**：
   - Dorian、Mixolydian 等
   - 现代爵士和电影音乐使用
   - 视频未提

5. **和声分析方法**：
   - 罗马数字标记法
   - 数字低音
   - 视频简化使用"5"和"6"

6. **现代爵士终止式**：
   - ii-V-I
   - tritone substitution
   - 视频未涉及

7. **录音参考**：
   - **巴赫圣咏 BWV 350**：4 种终止式集合
   - **莫扎特奏鸣曲 K.545**：典型完全终止
   - 视频未推荐参考。

8. **数字工具**：
   - **Practica Musica** 软件

   - 视频未提。
